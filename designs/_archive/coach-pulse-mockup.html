OPTION B · FULL-WIDTH UNIFIED
  • Permanently eliminates the height-mismatch gap
  • 4 live telemetry numbers always visible (ACWR, readiness, race, context)
  • Check-in inline — no separate card needed
  • Coach thought + chips + numbers in one scannable row
  • Biggest layout change but cleanest long-term structure
OPTION D · COACH DIRECT
  • Keeps 8+4 split (easier to implement, no grid change)
  • One very large "call" sentence — 2-second read
  • 3 big numbers right-aligned to the statement
  • Coach reasoning in smaller text below
  • More dramatic / immersive feel. Feels like a coach message
