Architecture
+ doctrine.

Deploy target

Runcino/web-v2 remains the Railway deploy target. The Faff/ repo is project HQ (design tokens, research, agents, fixtures); design + types mirror INTO Runcino/web-v2 per cutover. A future migration to a true Faff monolith is a Phase N+1 project.

Typography lock

Oswald 700 for display (hero verbs, card titles, stats). Inter 400/500/700 for body, labels, caps-tracked. Display recipe: letter-spacing: -0.015em; line-height: 0.86. Web auto-inherits via --f-display; iOS via .displayRecipe(size:).

11 day-states

easy · quality · long · rest · done_nailed · done_ease_off · niggle · sick · missed · race_week · new_user · skipped (new this session, 12 with skipped). Resolver is a pure function. State priority locked in glance-adapter.ts.

WeekStrip vocabulary

Closed 4-char set: EASY · INTS · TMPO · THRS · FART · QUAL · LONG · REST · XTRN · RACE · —. Backend emits the label; clients never compute it.

Skip Today doctrine

Active runner choice. Distinct from rest (planned), missed (passive), sick / niggle (health). Reversible. Stored in day_actions table. Plan continues tomorrow unchanged.

Simulator mode

?persona=<key> bypasses the DB lookup with a deterministic GlanceState. LLM briefing suppressed in simulator mode. Type-guarded URL handling — typos route to real data.

open + flagged

Known &
tracked.

  • iOS TTF bundling — Oswald TTFs ship in the repo at Resources/Fonts/; if you Archive without verifying Build Phases → Copy Bundle Resources includes them, Oswald falls back to system bold. Doesn't crash, just visual regression. Quick verify before each release.
  • Niggle / Sick states render in resolver but have no entry — Phase 19 deck designs the entry; code follows on direction call.
  • BodyGrid placeholders — WORKOUT BREAKDOWN + PHASE INTENT cards under /today render "Sprint 03" placeholders. Phase 14 (Poster workout-merge) addresses the breakdown.
  • day_actions migration 114 — applied to production DB via scripts/apply-114.mjs. iOS Skip GET degrades gracefully if it's ever missing (returns { skipped: false } instead of 500).
  • "Coach paused" card on /training, /races, /profile, /health — visual env-var indicator (COACH_PAUSED in Railway). Not a bug; tells you the LLM coach is off. Flip the env var to bring the coach back.
  • done_ease_off heuristic deferred — v1 routes all completed runs to done_nailed. Future: doneMi vs plannedMi ≥1.25× plus HR drift.
  • Persona sarah and marcus route to easy in the resolver because sick + niggle aren't yet entered via UI. Personas exist for the visual when the entry ships.
