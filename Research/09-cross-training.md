# 09 — Cross-Training Equivalents

How to substitute running with other modalities and calibrate equivalent training stimulus. Generic reference for an AI running coach.

## Definitions

- **Cross-training (XT):** Any non-running aerobic, strength, or mobility work performed to supplement or replace a planned run.
- **Modality:** A specific exercise mode (cycling, swimming, elliptical, etc.).
- **Carryover (transfer):** Degree to which fitness gained in modality A improves performance in modality B. Highest when biomechanics and dominant energy systems match.
- **Specificity principle:** Adaptations are specific to the muscles recruited, contraction patterns, and metabolic demands of the training stimulus. Cross-modal carryover is always partial.
- **Time-equivalency:** Substitute X minutes of running with X minutes of XT.
- **Intensity-equivalency:** Substitute by matching internal load (HR or RPE) regardless of duration.
- **Internal load:** Physiological cost (HR, RPE, lactate). Translates between modalities.
- **External load:** Mechanical output (pace, watts, distance). Does NOT translate between modalities.
- **HRmax-mode:** Sport-specific maximum heart rate, typically lower in non-running modalities.
- **DWR:** Deep water running (no foot contact, flotation belt). **SWR:** Shallow water running (foot contact, hip-to-chest depth).
- **Session-RPE (sRPE):** RPE 0–10 × duration (min) = arbitrary load units (Foster et al. 2001). Modality-agnostic.
- **MET:** Metabolic equivalent. 1 MET = 3.5 mL/kg/min VO2 (rest).

---

## Master Decision Rules

```
Rule 1 (specificity): Closer biomechanical match = higher run-fitness carryover.
        Rank: AlterG > Pool running (DWR) > Elliptical > Stair climber
              > Cycling > Rowing > Swimming
Rule 2 (intensity): Match internal load (HR, RPE), NOT pace or distance.
Rule 3 (HR offset):  In every non-running modality, HR at a given VO2 is LOWER
                     than running. Build sport-specific HR zones.
Rule 4 (duration):   Use time-equivalent substitution as the default
                     (1 min run -> 1 min XT) when intensity is matched.
        Exception: cycling load typically requires 1.3–1.5x the time
                   of the run it replaces to deliver equal stress.
Rule 5 (load):       sRPE = RPE(0-10) x minutes. Use to compare weekly load
                     across mixed running + XT weeks.
```

## Cycling

The most common XT modality. High aerobic ceiling, zero impact, but low biomechanical specificity (different muscle recruitment, no eccentric loading).

### Heart rate conversion

Cycling HR runs **5–15 bpm lower** than running at the same internal effort (∼5–8% on a percentage basis). The gap is largest for runners with little cycling experience and shrinks with cycling familiarity. Reasons: smaller active muscle mass, seated posture, no body-weight support cost.

```
Cycling HRmax  ≈ Running HRmax − 5 to 10 bpm
Cycling HR(zone)  ≈ Running HR(zone) − 8 to 12 bpm at submaximal intensities
```

Best practice: establish sport-specific zones via a cycling-specific test (e.g., 20-min FTP or ramp test) rather than reusing run zones.

### Power/FTP relationships

| Zone           | % cycling FTP | % running threshold pace (rough) | RPE 0–10 |
|----------------|---------------|----------------------------------|----------|
| Recovery       | <55%          | <70%                             | 1–2      |
| Endurance      | 56–75%        | 70–80%                           | 3–4      |
| Tempo          | 76–90%        | 84–88%                           | 5–6      |
| Threshold      | 91–105%       | 95–105%                          | 7–8      |
| VO2max         | 106–120%      | 105–115%                         | 9        |
| Anaerobic      | >120%         | >115%                            | 10       |

Note: in runners, VO2max is reached at roughly 115% of running threshold; in cyclists, ∼123% of FTP. The threshold-to-VO2max gap is wider in cycling.

### Time-equivalent substitution

| Run replaced              | Cycling substitute (HR-matched)   | Cycling substitute (time-matched) |
|---------------------------|-----------------------------------|-----------------------------------|
| 30 min easy run           | 45–50 min easy spin               | 30 min Z2                         |
| 60 min long easy          | 90 min Z2                         | 60 min Z2                         |
| 40 min tempo (20 min @ T) | 60 min ride (20 min @ FTP)        | 40 min ride (20 min @ FTP)        |
| 6×800m VO2 intervals      | 6×3 min @ 110–115% FTP, 3 min easy| —                                 |
| 90 min long run           | 2:00–2:30 ride                    | 90 min                            |

Common rule: **1 mile of running ≈ 10–15 minutes of moderate cycling** for aerobic stress, OR **3–4 miles of cycling ≈ 1 mile of running**. Both are coarse; HR/RPE matching is more reliable.

### Indoor vs. outdoor

- **Indoor (trainer/spin bike):** HR runs ∼5–10 bpm higher than equivalent outdoor power due to heat stress and lack of coasting. Adjust cooling.
- **Outdoor:** Includes stop-and-coast time. Add 15–25% to duration to get equivalent continuous load.

### Carryover

- **Carries over:** Aerobic capacity, mitochondrial density, capillarization, lactate clearance, cardiac output.
- **Does NOT carry over:** Eccentric quad/calf loading, ground-reaction-force tolerance, running economy, bone density, connective-tissue running-specific resilience.

## Aqua Jogging / Pool Running

Gold-standard injury substitute. Zero impact. The Wilber et al. (1996) study demonstrated trained runners maintained VO2max and 2-mile run performance over 6 weeks of DWR-only training (5×/week, alternating 30-min sessions at 90–100% VO2max with 60-min sessions at 70–75% VO2max). Subsequent reviews confirm fitness can be maintained for **up to 6–8 weeks** of full DWR substitution.

### Deep water vs. shallow water

| Attribute               | Deep water (DWR)              | Shallow water (SWR)               |
|-------------------------|-------------------------------|-----------------------------------|
| Foot contact            | None — flotation belt          | Yes — hip to chest depth          |
| VO2max relative to run  | ∼73% (lab max)                 | ∼90%                              |
| HRmax relative to run   | ∼86%                           | ∼89%                              |
| Spinal load             | Lowest (decompresses spine)    | Medium                            |
| Biomechanics            | Closer to running form         | Closer to running form, with impact |
| Best for                | Acute injury, no impact OK     | Late rehab, mild impact tolerated |

DWR is the default for injury rehab. SWR is a stepping-stone before returning to land running.

### HR calibration in water

Water immersion causes **immersion bradycardia**: HR at a given VO2 is **8–11 bpm lower** in water than on land. This is from hydrostatic pressure, cooler water temperature, and cardiac preload changes.

```
Aqua HR target ≈ Land run HR target − 10 bpm
HRmax in pool   ≈ Land HRmax − 10 to 15 bpm

Easy aqua zone:    65–75% land HRmax
Threshold aqua:    80–85% land HRmax
VO2 aqua:          90%+ land HRmax  (very hard to hit)
```

A common practical rule: target **10 bpm below your land easy run HR** for an easy aqua effort.

### Cadence

DWR cadence runs ∼65–80 strides/min — significantly lower than land running (∼170–180). Quick, high-knee turnover is required to keep HR elevated. Long, slow strokes drop HR fast.

### Sample DWR sessions

```
Easy/recovery:
  30–45 min steady, 70–75% land HRmax, RPE 3–4

Aerobic maintenance:
  60–90 min steady, 75–80% land HRmax, RPE 4–5

Tempo (threshold):
  Warm-up 10 min easy
  3×10 min @ 85% land HRmax / 2 min recovery
  10 min cool-down

VO2 intervals:
  10 min warm-up
  6–8 × 2 min very hard (max cadence) / 1 min easy
  10 min cool-down

Race-pace simulation:
  10 min warm-up
  10 × 1 min hard (10K effort) / 1 min easy
  10 min cool-down

Long aerobic:
  60–90 min with surges every 5 min (30 sec hard)
```

### Form cues

- Vertical torso, slight forward lean.
- High knee drive; quick foot turnover.
- Hands cup water, not flat — same arm swing as running.
- Belt fits above iliac crest, ribs above water.
- Avoid leaning back (reduces stride frequency and HR drops).

### Spinal effects

DWR reduces spinal compression compared to land running (loss of stature ∼2.9 mm DWR vs. 4.6 mm land run vs. 5.5 mm SWR). Useful adjunct after high-volume run blocks.

## Elliptical

Most-similar non-running modality available in commercial gyms. Bilateral, weight-bearing (no impact spike), continuous foot contact. When intensity is matched, VO2 and HR responses are nearly identical to treadmill running.

### Intensity matching

```
Elliptical HR ≈ Run HR − 5 to 10 bpm at the same RPE
1:1 time substitution at matched RPE delivers comparable aerobic stimulus
```

A 12-week RCT (Mercer et al. and others) showed equivalent physiological adaptations between elliptical and treadmill training when matched for volume and intensity.

### Setup that maximizes specificity

- Resistance: low-moderate (5–8/20). High resistance recruits quad-dominant patterns.
- Stride: longest setting available (closer to running stride).
- Incline/ramp: moderate-to-high (mimics running mechanics, reduces quad bias).
- Hands off rails for at least part of the session — engages core, raises HR.
- Cadence: 160–180 SPM mirrors running cadence.

### Sample elliptical sessions

```
Easy run substitute:
  30–60 min @ RPE 3–4, low resistance, hands off

Tempo:
  10 min warm-up
  2×15 min @ RPE 6–7, moderate resistance / 3 min easy
  10 min cool-down

VO2 intervals:
  10 min warm-up
  5×4 min @ RPE 8–9 / 3 min easy
  10 min cool-down
```

## Rowing

Whole-body, posterior-chain dominant. Aerobic carryover real but lower per minute than cycling/elliptical (form fatigue often limits intensity before cardiovascular limit).

### Conversion

```
HR offset:    ∼5 bpm lower than running at matched RPE
Time ratio:   1:1.0–1.1 (rowing slightly easier per minute,
              especially as form fatigues)
2K row time   correlates moderately with 5K run time at amateur level;
              poorly at elite level
```

### Best uses

- Posterior-chain strengthening (hamstrings, glutes, erectors).
- Off-season aerobic with upper-body inclusion.
- Limited to 30–45 min sessions for most runners (form fatigue).

### Sample sessions

```
Aerobic:
  30–45 min steady @ 22–24 spm, RPE 4–5

Threshold:
  4×8 min @ RPE 7 / 2 min easy

VO2:
  8×500m hard / 2 min rest
  or 5×3 min hard / 2 min rest
```

## Stair Climbing / StairMaster

Vertical work. Higher per-minute energy cost than flat running at moderate speeds, but mechanical fatigue often caps duration. Strong glute and calf demand.

### Intensity

| Effort        | StairMaster steps/min | RPE |
|---------------|------------------------|-----|
| Easy          | 50–70                  | 3–4 |
| Moderate      | 70–95                  | 5–6 |
| Hard          | 95–115                 | 7–8 |
| Max sustained | 115+                   | 9   |

### Substitution

- 1 min stair climbing ≈ 1.0–1.2 min easy/moderate running for aerobic stress.
- Caps out at ∼45–60 min before muscular fatigue limits HR.
- Use as a hill-strength substitute or shorter aerobic session.

### Best uses

- Hill-running simulation when terrain unavailable.
- Glute/calf strength + aerobic combo.
- Limited-time conditioning.

## Hiking and Walking

### Walking

Low metabolic stress at flat-ground pace. Aerobic carryover is small for trained runners; useful for recovery, deload, or returning from total layoff.

```
Walking ≈ recovery / Z1 only for trained runners
3 min walking ≈ 1 min easy running for cardiovascular load (flat)
```

### Hiking (graded terrain)

On steep grades hiking approaches running's metabolic cost. At 30°+ inclines walking is more economical than running and recruits eccentric quad and posterior chain similar to downhill running. Hiking loaded (pack) further raises load.

```
Flat hiking:        Z1, count as recovery
Rolling hiking:     Z2 stimulus, ∼0.7x equivalent run time
Steep uphill hike:  Z2–Z3 stimulus, near 1:1 substitution at HR-matched effort
Loaded hike (pack): Add ∼10–15% intensity per 10% bodyweight load
```

### Substitution rule

- For trained runners, count rolling hikes as **70% of equivalent run time** for aerobic load.
- Steep uphill hikes count 1:1 if HR is matched.

## Skiing (Nordic / Cross-Country)

Highest VO2max sport on record (elite skiers commonly 80+ mL/kg/min). Large active muscle mass (upper + lower body) yields exceptional aerobic carryover. Cross-country skiing builds aerobic engine effectively for runners during winter.

### Notes

- Classical and skating both work; skating has slightly lower running carryover (different lower-limb pattern).
- Roller skiing/SkiErg approximates the modality off-snow.
- Elite skiers achieve **higher VO2max in running than in skating**, suggesting running stimulus is harder to fully replicate even with skiing's large active muscle mass.

### Substitution

```
Cross-country ski (steady)  ≈ 1:1 time match with easy run at matched HR
SkiErg interval session     ≈ rowing-equivalent stimulus, more upper body
HR offset                   ≈ 0–5 bpm vs. running (very close)
```

## Strength Training as Cross-Training

See Task 7 (Strength Training) for full prescription. Brief integration:

- Strength is a complement, not a substitute, for aerobic work.
- 2×/week heavy lower-body sessions improve running economy and injury resistance.
- During injury layoff: shift to 3×/week, add isolation work for the injured area's adjacent musculature.
- Quantify session load via sRPE; do not equate to run minutes.

## Yoga and Pilates

### Yoga

- Primary effect: mobility, recovery, parasympathetic activation.
- Aerobic stimulus minimal except in vigorous vinyasa/power yoga.
- Use as recovery-day or active-rest substitute.
- Power yoga can substitute for an easy 30 min if HR averages Z1–low Z2.

### Pilates

- Primary effect: core, hip stability, postural control.
- A 12-week RCT (PLoS One 2018) showed Pilates intervention improved 5K time in trained runners (∼25:39 → 23:13 post-intervention) — suggests transfer via running economy and injury resistance.
- Reformer pilates produces higher strength stimulus than mat.

### Substitution rule

- Yoga and pilates do NOT substitute for aerobic running sessions on a 1:1 basis.
- Use them as **additions** on easy/recovery days, or as the only session on planned active-rest days.

## When to Use Cross-Training

| Trigger                        | Default XT prescription                                  |
|--------------------------------|----------------------------------------------------------|
| Acute injury (no run possible) | DWR or AlterG, full volume substitution, 6–8 wk maintains |
| Niggle / managed pain          | Elliptical or DWR for 1–2 sessions; resume run gradually |
| Post-race recovery (≤7 days)   | Walking, yoga, easy spin only                            |
| Heavy training block recovery  | 1 swim or easy bike replaces 1 easy run                  |
| Severe weather                 | Treadmill > elliptical > bike (specificity ranking)      |
| Schedule constraint            | Bike commute as easy-run substitute                      |
| Heat/altitude acclimation      | Pool running for cooling, indoor bike for sustained work |
| Off-season aerobic base        | Add 1–2 XT sessions to running week                      |
| Beginner / overuse risk        | 2 runs + 2–3 XT sessions per week                        |

## How Much Cross-Training to Substitute

### When fully replacing running (injured)

| Run hours/week (lost) | XT hours/week (replacement) |
|-----------------------|------------------------------|
| 3                     | 3.0                          |
| 5                     | 5.0–5.5                      |
| 8                     | 8.0–9.0                      |
| 10                    | 10.0–11.0                    |
| 12+                   | 12.0–13.0                    |

Match time directly when intensity is matched. Add 5–10% to compensate for slightly lower per-minute stimulus in non-running modalities.

### Distribution within injury weeks

```
70–80% of XT volume:  Easy/aerobic (matches lost easy mileage)
15–25%:               Threshold/tempo work
5–10%:                VO2max-style intervals
```

Mirror the intensity distribution of the running plan being replaced.

### When supplementing running

| Run base    | Add XT (week)            | Notes                                |
|-------------|---------------------------|--------------------------------------|
| 2–3 runs    | 2–3 XT sessions, 30–60 min | Build aerobic base without injury    |
| 4–5 runs    | 1–2 XT sessions, 30–60 min | Recovery + cross-stimulus            |
| 6+ runs     | 0–1 XT, 30–45 min         | Most volume should be running        |

Rule: never let XT compromise key running sessions. XT replaces easy runs first, hard runs last.

## Maintaining Run Fitness During Forced Layoffs

### Detraining timeline (no training at all)

| Time off        | VO2max change | Other changes                                          |
|-----------------|---------------|--------------------------------------------------------|
| 0–7 days        | ≈ 0% (often supercompensation in fatigued athletes) | Plasma volume −5%; glycogen ↑ |
| 10–14 days      | −4 to −7%     | Lactate threshold begins to drop                       |
| 2–3 weeks       | −6 to −9%     | Mitochondrial density drops, capillarization slowly    |
| 4 weeks         | −10 to −14%   | Clear performance decline                              |
| 6–9 weeks       | −15 to −19%   | Significant decline in trained athletes                |
| 11+ weeks       | −20 to −26%   | Major detraining; long return required                 |

### With cross-training maintenance

DWR-only protocols (Wilber et al.) maintained VO2max and 2-mile time over 6 weeks. Multiple subsequent studies confirm:

- **Up to 4 weeks:** XT-only maintains race fitness with minimal loss.
- **4–8 weeks:** Most fitness maintained; running-specific economy and ground-contact resilience decline.
- **8+ weeks:** Aerobic capacity preserved, but expect a 2–4 week return-to-run period to rebuild running economy and connective tissue.

### Priorities during layoff (in order)

1. **Volume** — match lost weekly running hours in XT.
2. **Intensity distribution** — preserve the polarized mix.
3. **Frequency** — daily or near-daily aerobic touches.
4. **Strength** — 2–3×/week maintains neuromuscular fitness.
5. **Specificity** — choose modality closest to running (pool > elliptical > bike).

## Time-Equivalency vs. Intensity-Equivalency

| Approach            | Method                                | When to use                          |
|---------------------|---------------------------------------|--------------------------------------|
| Time-equivalent     | Match minutes, ignore intensity drift | Quick coarse substitution            |
| Intensity-equivalent| Match HR/RPE, accept any duration     | Replacing specific quality sessions  |
| Load-equivalent (sRPE) | Match RPE × minutes               | Weekly load balancing across modalities |

### Decision rule

```
IF replacing an easy run THEN use time-equivalent + HR-matched (default)
IF replacing a quality session (tempo, intervals) THEN intensity-equivalent
IF tracking weekly load across mixed modalities THEN sRPE
```

### Session-RPE formula

```
sRPE_load = RPE(0–10) × duration(minutes)

Example:
  60-min easy run @ RPE 3   = 180 AU
  75-min easy bike @ RPE 3  = 225 AU
  45-min tempo run @ RPE 7  = 315 AU
  60-min tempo bike @ RPE 7 = 420 AU
```

Foster's session-RPE has been validated against HR-based load measures across cycling, running, rowing, and team sports. Accepted as the simplest cross-modal load metric.

## HR-Based vs. Pace-Based Cross-Training

### What translates

| Metric           | Cross-modal validity | Notes                                    |
|------------------|----------------------|------------------------------------------|
| Pace / distance  | Does not translate   | Each modality has its own external load  |
| Power (W)        | Does not translate   | Cycling/rowing watts ≠ running effort    |
| Heart rate       | Translates with sport-specific zones | Build modality-specific HRmax |
| RPE              | Translates well      | Best universal metric                    |
| sRPE (load)      | Translates well      | Use for weekly load comparisons          |
| Lactate          | Translates           | Equipment intensive                      |

### Practical implications

- Don't try to convert "10 miles of cycling = X miles of running" for prescription. Use HR or RPE.
- Build sport-specific HR zones (LTHR test or ramp test in each modality) for accurate intensity targeting.
- For VO2 sessions, RPE 9–10 in any modality approximates VO2 work.
- Threshold work: RPE 7–8, durable for 30–60 min.

### HR zone offsets cheat-sheet

| Modality     | HR offset from running zones |
|--------------|------------------------------|
| AlterG ≥80%  | 0 bpm                        |
| Treadmill    | 0 bpm                        |
| Elliptical   | −5 to −8 bpm                 |
| Stair climber| −3 to −5 bpm                 |
| Rowing       | −5 bpm                       |
| Nordic ski   | 0 to −5 bpm                  |
| Cycling      | −5 to −15 bpm                |
| Swimming     | −10 to −15 bpm               |
| DWR          | −8 to −11 bpm (immersion bradycardia) |
| SWR          | −7 bpm                       |

## Combining Running and Cross-Training in the Same Week

### Sequencing principles

```
1. Hard XT counts as a quality day. Never stack with a hard run.
2. Easy XT pairs well with: morning of an easy run, or PM of an AM run.
3. Place XT 24–48 h after a hard run (recovery/aerobic stimulus).
4. Do not place hard XT 24 h before a hard run (compromises both).
5. Long XT (>90 min) replaces, not supplements, a long run.
6. Strength: ideally same day as hard run (post-run) OR ≥6 h gap.
```

### Sample weeks

**Healthy runner — 5 runs + 2 XT (recovery focus):**

```
Mon  Easy run 45 min
Tue  Threshold run + strength
Wed  Easy spin 60 min (recovery aerobic)
Thu  Easy run 50 min
Fri  Yoga / mobility 30 min
Sat  VO2 intervals
Sun  Long run
```

**Injured runner — 0 runs, full XT replacement:**

```
Mon  DWR easy 60 min + strength
Tue  Bike threshold (2x20 min)
Wed  Elliptical easy 60 min
Thu  DWR intervals (10x2 min)
Fri  Yoga + strength
Sat  Bike VO2 (5x4 min)
Sun  DWR long 90 min
```

**Mixed modality — 4 runs + 3 XT (volume add for aerobic-limited runner):**

```
Mon  Easy run 45 min
Tue  Bike threshold 60 min
Wed  Easy run 50 min + strength
Thu  Easy elliptical 45 min
Fri  Off / yoga
Sat  Long run
Sun  Easy bike 75 min
```

### Hard-day pairing rules

| Day N session         | Day N+1 OK?       | Day N+1 avoid              |
|-----------------------|-------------------|----------------------------|
| Hard run (intervals)  | Easy XT, easy run | Hard XT, hard run          |
| Hard XT (bike VO2)    | Easy run, easy XT | Hard run                   |
| Long run              | Easy XT only      | Any hard session           |
| Tempo run             | Recovery only     | Hard XT, intervals         |
| Strength (heavy LB)   | Easy run, easy XT | Hard run, hard XT          |

### 80/20 rule with XT

The polarized intensity distribution (∼80% easy, ∼20% hard) should be preserved when combining modalities. Count XT minutes toward both buckets the same way you would running minutes.
