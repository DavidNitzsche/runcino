# 04 — Workout Vocabulary and Structures

A reference library of named running workouts. Use this doc to look up the prescribed structure of any workout name. Generic prescriptions only — paces are expressed in zone terms (E/M/T/I/R) or as percentages of physiological/race-pace anchors, not specific clock paces.

## Pace zone shorthand

This doc references paces by Daniels' single-letter zones plus race-pace anchors. Definitions:

| Code | Name | Physiological target | % VO2max | % HRmax | Race-pace anchor |
|------|------|----------------------|----------|---------|------------------|
| E | Easy | Aerobic base, capillary density, mitochondrial volume | 59–74% | 65–79% | ~MP + 60–90 s/mi (1–2 min/mi slower than MP) |
| M | Marathon | Marathon-specific aerobic | 75–84% | 80–89% | Goal MP |
| T | Threshold | Lactate threshold (LT2, ~4 mmol/L) | 86–88% | 88–90% | ~1-hour race pace, ≈HM pace for sub-elite, ~10 mi pace for elites |
| ST | Sub-threshold | Below LT2, near LT1 (~2.5–3.5 mmol/L) | 80–86% | 84–88% | ~10–15 s/mi slower than T |
| I | Interval | VO2max | 95–100% | 95–100% | ~3K–5K race pace |
| R | Repetition | Speed/economy/anaerobic | n/a (above VO2max) | maxes out | ~mile to 800m race pace |
| HM | Half-marathon pace | Slightly below T | 84–86% | 87–90% | Current HM race pace |
| MP | Marathon pace | High aerobic | 75–84% | 80–89% | Current MP |
| 10K | 10K race pace | Just above T | 88–92% | 90–93% | Current 10K |
| 5K | 5K race pace | At/near VO2max | 95–100% | 95–98% | Current 5K |
| 3K | 3K race pace | Above VO2max | 100–105% | maxes out | Current 3K |

LT1 = first lactate inflection (~2 mmol/L), aerobic threshold. LT2 = second inflection (~4 mmol/L), lactate threshold. VO2max = maximum oxygen uptake. MP/HM/10K/5K/3K = current race pace at that distance.

## 1. Recovery runs

| Field | Prescription |
|-------|--------------|
| Purpose | Active recovery between hard sessions; promote circulation and waste clearance without adding training stress |
| Physiological target | Capillary blood flow; minimal mitochondrial demand |
| Pace | Slower than E. ~MP + 90+ s/mi, or 60–70% HRmax, or "easier than easy" |
| Duration | 20–45 min; capped to keep total stress low |
| Frequency | Day after a hard session, or as a second daily run |
| Volume role | Should not exceed ~10–15% of weekly mileage |
| When in cycle | All phases. More common in high-volume base and race-specific phases when key sessions are dense |
| Contraindications | Replace with full rest if HRrest is elevated >7 bpm above baseline, or if cumulative fatigue is unmanaged |
| Variations | Recovery shakeout (15–20 min), double-day recovery (AM or PM tag-on), elliptical/cycle cross-train substitute |

## 2. Easy / general aerobic runs

| Field | Prescription |
|-------|--------------|
| Purpose | Build aerobic base; develop capillary density, mitochondrial enzymes, slow-twitch fiber endurance |
| Physiological target | Heart adaptations (stroke volume), Type I fiber adaptations |
| Pace (Daniels E) | 59–74% VO2max, 65–79% HRmax. Conversational. Roughly MP + 60–90 s/mi |
| Pace (Pfitzinger general aerobic) | 15–25% slower than MP; 70–81% HRmax; 62–75% HRR |
| Duration | 30–75 min typical; up to 90 min for high-mileage runners |
| Frequency | 3–5 days per week; bulk of weekly volume |
| Volume role | 70–85% of weekly mileage |
| When in cycle | All phases. Volume rises in base, holds steady through specific |
| Contraindications | Drift toward M pace defeats the purpose — keep effort honest |
| Variations | Pure easy, general aerobic (slightly faster end), with strides appended |

---

## 3. Medium-long runs

| Field | Prescription |
|-------|--------------|
| Purpose | Extend aerobic stimulus mid-week without long-run recovery cost; bridge the gap between daily runs and the weekly long run |
| Physiological target | Glycogen depletion, fat oxidation at longer durations |
| Distance (Pfitzinger) | 11–15 miles |
| Pace | Same as long-run pace: E to low M effort. ~15–20% slower than MP |
| Frequency | 1×/week during marathon and HM specific phases |
| Volume role | Often the second-longest run of the week |
| When in cycle | Pfitzinger marathon and HM plans use it through base and specific phases |
| Contraindications | Don't run too hard — it should not compete with the long run for recovery |
| Variations | Plain medium-long, medium-long with strides, medium-long with embedded T segment (advanced) |

## 4. Long runs

The long run is the cornerstone weekly session in distance training. Variants below differ by purpose and pace structure.

### 4.1 Long-run family overview

| Variant | Distance/duration | Pace structure | Primary stimulus | Typical placement |
|---------|-------------------|----------------|------------------|-------------------|
| Base long run | 90 min – 2:30; up to 22 mi for marathoners | E throughout | Aerobic capacity, glycogen storage, fat oxidation | Base phase, all distances |
| Progression long run | 12–22 mi | Start E, finish M to T | Late-run pace tolerance | Specific phase |
| Marathon-pace long run | 14–22 mi | Easy warmup + 8–16 mi at MP | Marathon-specific economy | Marathon specific phase |
| Fast finish long run | 12–18 mi | Easy bulk + last 2–6 mi at MP or faster | Late-stage pace skill, mental toughness | Marathon and HM specific |
| Dress rehearsal long run | 18–22 mi | Easy + MP segments; full race-day setup | Fueling/gear/timing rehearsal | 3 weeks pre-marathon |

### 4.2 Base long run

| Field | Prescription |
|-------|--------------|
| Purpose | Build aerobic capacity, glycogen storage, and connective-tissue durability |
| Pace | E pace. ~55–75% of 5K pace; 60–75% HRmax |
| Duration | 90 min minimum for endurance benefit; cap at ~25–30% of weekly mileage |
| Distance | 10–22+ mi for marathoners; 8–14 mi for 5K/10K |
| Frequency | Weekly |
| Volume role | 20–25% of weekly mileage typical |
| When in cycle | Base phase: build duration. Specific phase: hold or replace with structured variants |
| Contraindications | Beginners should cap at 20% of weekly mileage; avoid back-to-back hard days |

### 4.3 Progression long run

| Field | Prescription |
|-------|--------------|
| Purpose | Train the neuromuscular and metabolic skill of running faster while fatigued |
| Structure | First 1/3 to 1/2 at E pace, middle at strong E or M, final 1/4 to 1/3 at M to T |
| Example (16 mi) | 6 mi E + 6 mi M + 4 mi T |
| Recovery | None — continuous |
| Frequency | Every 2–3 weeks in specific phase |
| Contraindications | Skip if accumulated fatigue is high; don't pair with other quality work in same week |

### 4.4 Marathon-pace long run

| Field | Prescription |
|-------|--------------|
| Purpose | Marathon-specific stimulus; rehearse race pace under fatigue |
| Structure | Easy warmup (2–4 mi) + 8–16 mi at MP + optional easy cooldown |
| Common dose | 14–18 mi total with 10–14 mi at MP |
| Pace | MP exactly — not faster |
| Frequency | Every 2–3 weeks during marathon specific phase |
| When in cycle | 6–10 weeks out from goal marathon |
| Contraindications | Don't run if calf/Achilles/hip is flagging — MP is high-stress for soft tissue |
| Variations | MP locked in second half (harder), MP from the start (hardest), MP with surges |

### 4.5 Fast finish long run

| Field | Prescription |
|-------|--------------|
| Purpose | Train ability to find pace late; mental rehearsal of "the last 10K" |
| Structure | Bulk at E (1–2 min/mi slower than MP), final 2–6 mi at MP or slightly faster |
| Example | 16 mi total: 12 mi E, last 4 mi at MP; or 14 mi: last 3 mi at MP, last 1 mi sub-MP |
| Recovery | None |
| Frequency | Every 2–3 weeks |
| When in cycle | Specific phase, marathon and HM |

### 4.6 Dress rehearsal long run

| Field | Prescription |
|-------|--------------|
| Purpose | Final equipment, fueling, and timing rehearsal |
| Distance | 18–22 mi (marathon); 12–14 mi (HM) |
| Structure | Race-day breakfast, race-day kit, race-day fueling intervals; segments at MP |
| Pace | Easy bulk + 2–3 segments at MP (4–8 mi total at MP) |
| When in cycle | 3 weeks pre-marathon; before taper begins |
| Contraindications | Not a fitness builder — keep effort controlled. If injury threat, skip MP segments |

## 5. Threshold workouts

Threshold work targets LT2 (lactate threshold) and the band just below it. Goal: extend the velocity at which lactate clearance matches production.

### 5.1 Threshold family overview

| Workout | Volume | Pace | Recovery | Total at-pace |
|---------|--------|------|----------|---------------|
| Continuous tempo | 3–8 mi continuous | T | None | 20–40 min |
| Cruise intervals (Daniels) | 3–6 × 1 mi or 2–4 × 2 mi | T | 1 min per mi of work | 4–8 mi |
| Sub-threshold intervals (Norwegian) | 5–10 × 1K or 4–6 × 2K | ST (10–15 s/mi slower than T) | 60–90 s | 5–10 K |
| Long tempo | 8–12 mi continuous | Slightly slower than T (HM-ish) | None | 8–12 mi |

### 5.2 Continuous tempo (4–8 mi at threshold)

| Field | Prescription |
|-------|--------------|
| Purpose | Raise lactate threshold; build mental tolerance to sustained discomfort |
| Pace | T pace (86–88% VO2max, 88–90% HRmax). "Comfortably hard" — sustainable for ~1 hour in a race |
| Duration | 20 min minimum for stimulus; 20–40 min sweet spot |
| Distance | 3–8 mi |
| Recovery | None — single block |
| Warmup/cooldown | 2–3 mi E each side |
| Frequency | 1×/week or alternating with cruise intervals |
| When in cycle | Base into specific phase; backbone of HM and marathon training |
| Contraindications | Skip if HR/perceived effort elevated — pace will lie. Replace with cruise intervals on bad days |
| Variations | Cutdown tempo (start MP, finish T), wave tempo (alternate ±10 s/mi around T) |

### 5.3 Cruise intervals (Daniels)

| Field | Prescription |
|-------|--------------|
| Purpose | Accumulate more time at T than a single tempo allows |
| Pace | T pace |
| Structure | 3–6 × 1 mi with 1 min jog, or 2–4 × 2 mi with 2 min jog |
| Recovery | 1 min jog per mile of work segment |
| Total volume at pace | 4–8 mi (Daniels: cap T-pace at 10% of weekly mileage) |
| Warmup/cooldown | 2–3 mi E each side |
| Frequency | 1×/week |
| When in cycle | All phases except deepest base |
| Contraindications | Lengthening rest changes the workout — keep recoveries short |
| Variations | Cruise + threshold (e.g., 4 × 1 mi + 2 mi continuous), cruise pyramid (1-2-1-2-1) |

### 5.4 Sub-threshold (Norwegian) intervals

| Field | Prescription |
|-------|--------------|
| Purpose | Accumulate large weekly threshold volume without the systemic cost of tempo |
| Pace | ST: ~10–15 s/mi slower than T; lactate target 2.5–3.5 mmol/L |
| Structure (single threshold) | 5–10 × 1K, or 4–6 × 2K, or 4–6 × 6 min |
| Structure (double threshold) | AM: 4–6 × 1K or 5 × 6 min ST; PM: 8–10 × 1K or 6 × 2K slightly faster ST |
| Recovery | 60–90 s jog (short, by design) |
| Total volume | 5–10 K at ST per session; 12–20 K total threshold across a double day |
| Frequency | 2×/week (singles) or 1 double-day every 4–7 days |
| When in cycle | Base and specific; sustainable year-round at single-threshold dose |
| Contraindications | Requires honest pace discipline — going too hard collapses the model. Doubles require high training age and low life stress |
| Variations | Hill ST intervals (Ingebrigtsen variant), 1000m at ST descending across the set |

### 5.5 Long tempo (8–12 mi)

| Field | Prescription |
|-------|--------------|
| Purpose | Marathon-specific aerobic stress; HM race rehearsal |
| Pace | Slightly slower than T — typically HM pace to T-minus-5 s/mi |
| Distance | 8–12 mi continuous |
| Recovery | None |
| Warmup/cooldown | 1–2 mi E each side |
| Frequency | Every 2–3 weeks in specific phase |
| When in cycle | HM and marathon specific; not a base-phase staple |
| Contraindications | High accumulated fatigue cost — schedule at least 2 easy days after |
| Variations | 2 × 5 mi at HM with 3 min jog (split version), 8 mi MP + 2 mi T (combo) |

## 6. VO2max workouts

VO2max work targets max aerobic power. Daniels' rule: each interval should be 3–5 min long; total at-pace volume ≤ 8% of weekly mileage; recovery roughly equals interval duration.

### 6.1 VO2max family overview

| Workout | Reps × distance | Pace | Recovery | Total at-pace |
|---------|-----------------|------|----------|---------------|
| Mile repeats (3K/5K) | 3–6 × 1 mi | I (5K) to slightly slower | 2:30–4:00 jog (≈ rep time) | 3–6 mi |
| 1200m repeats | 4–6 × 1200 | I | 2–3 min jog | 4.8–7.2 K |
| 1000m repeats | 5–8 × 1K | I | 2–3 min jog (≈ rep time) | 5–8 K |
| 800m repeats | 6–10 × 800 | I | 2–3 min jog (≈ rep time) | 4.8–8 K |
| 600m repeats | 8–12 × 600 | I to slightly faster | 2–3 min jog | 4.8–7.2 K |
| 400m repeats | 8–16 × 400 | 3K–5K | 90 s – 2 min jog | 3.2–6.4 K |
| Yasso 800s | 4–10 × 800 | Time-prediction-based | Equal time jog | 3.2–8 K |

### 6.2 Mile repeats (3–6 × 1 mile)

| Field | Prescription |
|-------|--------------|
| Purpose | Develop VO2max and 5K/10K-specific endurance; build tolerance to sustained hard effort |
| Pace | I pace (≈5K race pace). For HM training, slow to slightly faster than HM |
| Reps | 3 (entry), 4–5 (typical), 6 (advanced) |
| Recovery | Jog rest ≈ rep time. 3:00–5:00 typical. Daniels: ≤ rep time |
| Total volume | Cap at 8% weekly mileage |
| Warmup/cooldown | 2–3 mi E + drills + 2–4 strides; 1–2 mi E cooldown |
| Frequency | Every 7–10 days |
| When in cycle | Mid-cycle through specific phase for 5K/10K; base/specific for HM |
| Contraindications | Avoid in week of taper (last hard session ≥10 days out) |
| Variations | Mile repeats descending (each rep faster), mile repeats at HM pace (more reps, less rest), broken miles (e.g., 1200+400 with 30 s jog) |

### 6.3 1000m repeats

| Field | Prescription |
|-------|--------------|
| Purpose | Classic VO2max workout — ideal interval duration (3–4 min) for maxing out aerobic power |
| Pace | I pace |
| Reps | 5–8 |
| Recovery | 2:00–3:00 jog (≈ rep time, 200–400 m jog) |
| Total volume | 5–8 K at I |
| Warmup/cooldown | Standard track warmup (see §17) |
| Frequency | Weekly during VO2max block |
| When in cycle | Specific phase for 5K/10K; mid-cycle for HM |
| Variations | 5 × 1K cutdown, 4 × 1K + 4 × 400, 3 × (2 × 1K) with longer set rest |

### 6.4 800m repeats

| Field | Prescription |
|-------|--------------|
| Purpose | VO2max with slightly more turnover than 1Ks; classic 5K specific |
| Pace | I pace (≈ 3K–5K) |
| Reps | 6–10 |
| Recovery | 2:00–3:00 jog (≈ rep time) |
| Total volume | 4.8–8 K at I |
| Frequency | Weekly during VO2 block |
| Contraindications | Avoid running first reps too fast — first rep should not be the fastest |
| Variations | 8 × 800 at 5K pace (classic), Yasso 800s (see §6.7), 800s descending |

### 6.5 600m repeats

| Field | Prescription |
|-------|--------------|
| Purpose | Bridge between true VO2 and faster turnover; useful for milers and 5K runners |
| Pace | I to slightly faster than I (≈ 3K pace) |
| Reps | 8–12 |
| Recovery | 2:00–3:00 jog or 400m jog |
| Total volume | 4.8–7.2 K |
| Frequency | Every 7–10 days |
| When in cycle | Late base / specific |

### 6.6 400m repeats

| Field | Prescription |
|-------|--------------|
| Purpose | Edge between VO2 and speed; develops anaerobic component and turnover |
| Pace | 3K–5K race pace (faster than pure I) |
| Reps | 8–16 |
| Recovery | 90 s – 2 min jog (or 200–400 m jog) |
| Total volume | 3.2–6.4 K |
| Note | Daniels: pure 400s alone are inefficient for VO2max because it takes ~2 min to elicit VO2max. Use 400s for speed/economy or as part of a longer set |
| Frequency | Weekly in 5K specific phase |
| Variations | 16 × 400 at 5K (classic), 12 × 400 alternating I/R, 4 × (4 × 400) with longer set rest |

### 6.7 Yasso 800s

| Field | Prescription |
|-------|--------------|
| Purpose | Marathon prediction workout (Bart Yasso, 1994); VO2max stimulus |
| Pace formula | Time per 800 in min:sec = goal marathon time in hr:min. Example: 3:30 marathon → 3:30 per 800m |
| Reps progression | Build from 4 → 10 across cycle. Final session: 10 × 800 |
| Recovery | Jog the same time it took to run the 800 (equal-time jog, ~400m) |
| Pace zone | ≈ 5K to 10K race pace for typical sub-3:30 marathoners |
| Frequency | 1×/week during build |
| When in cycle | Last benchmark 10–14 days before goal marathon |
| Prediction caveat | Modern data: Yasso 800s overpredict for slower marathoners and underpredict for faster ones. VDOT-based prediction from recent races has lower error |
| Contraindications | Don't substitute for marathon-specific work — VO2max session, not MP-specific |

## 7. Speed / economy workouts

Short, fast, full-recovery work to develop neuromuscular coordination, running economy, and stride mechanics. Daniels: cap R pace at 5% of weekly mileage.

### 7.1 Speed family overview

| Workout | Distance | Pace | Recovery | Total |
|---------|----------|------|----------|-------|
| Strides | 50–100 m | Mile to 5K race pace; relaxed sprint | Full walk/jog | 4–8 reps |
| Hill sprints | 8–15 s | Max effort uphill | Walk-down full recovery | 6–12 reps |
| 200m repeats | 200 m | R pace (≈ mile pace) | 200m jog | 8–12 reps |
| 100m repeats | 100 m | R to faster | 100m walk/jog | 8–16 reps |

### 7.2 Strides

| Field | Prescription |
|-------|--------------|
| Purpose | Neuromuscular activation; recruit fast-twitch fibers; refine form on easy days |
| Distance | 50–100 m or 15–30 s each |
| Pace | Accelerate to mile-to-5K race pace; ~85–95% max effort, relaxed |
| Reps | 4–8 |
| Recovery | Full walk-back or 60–90 s jog — no fatigue between strides |
| Frequency | 2–4×/week |
| Placement | End of an easy run, mid-warmup before a workout, or standalone day |
| When in cycle | All phases — never stop doing strides |
| Contraindications | Not a workout — back off if form deteriorates |
| Variations | Hill strides (slight uphill, 10–15 s), in-and-out strides (accelerate-cruise-accelerate), strides on grass for impact-sensitive runners |

### 7.3 Hill sprints

| Field | Prescription |
|-------|--------------|
| Purpose | Build leg power, tendon stiffness, neuromuscular drive; injury-resilient speed work |
| Grade | Steep (8–15%); steepest hill manageable with form |
| Duration | 8–15 s |
| Pace | Max effort, all-out |
| Reps | Start 4–6, build to 8–12 |
| Recovery | Walk down — full recovery (2–3 min). Heart-rate based: until normal breathing returns |
| Frequency | 1–2×/week |
| When in cycle | Year-round. Especially valuable in base phase |
| Contraindications | Not for first-month-back runners; require base of easy running |
| Variations | Single-leg bounding hills, alternating sprint/stride hills |

### 7.4 200m repeats

| Field | Prescription |
|-------|--------------|
| Purpose | Speed development; running economy; race-finish kick |
| Pace | R pace (≈ mile race pace) |
| Reps | 8–12 |
| Recovery | 200m jog (full recovery, equal-distance jog) |
| Total | 1.6–2.4 K at R |
| Warmup | Drills + 4 strides |
| Frequency | Weekly during speed block; 1× every 2 weeks otherwise |
| When in cycle | Base, late specific, taper week |
| Contraindications | Cap at 5% weekly mileage; don't shorten the rest |
| Variations | 200-200-400 cycles (Daniels), 200m descending, 200m at mile pace into 200m float |

### 7.5 100m repeats

| Field | Prescription |
|-------|--------------|
| Purpose | Pure speed; turnover; mechanics |
| Pace | R pace or faster |
| Reps | 8–16 |
| Recovery | 100m walk or jog — full recovery |
| Frequency | 1×/week or as part of warmup for longer sessions |
| When in cycle | All phases |
| Variations | 10 × 100 alternating fast/relaxed, 100m fly-ins |

## 8. Hill workouts

Hills produce speed-like stimulus with reduced impact. Categories below by duration.

### 8.1 Hill family overview

| Workout | Duration | Grade | Pace | Reps | Recovery |
|---------|----------|-------|------|------|----------|
| Short hill repeats | 10–30 s | 4–7% | Strong, controlled (~95% effort) | 8–16 | Walk/jog down |
| Medium hill repeats | 60–90 s | 4–6% | 5K–10K effort | 6–10 | 2–3 min jog down |
| Long hill repeats | 3–5 min | 3–5% | T to 10K effort | 4–8 | Equal jog down |
| Hill circuit (Lydiard) | ~1.9 mi loop | Mixed | Sequence of efforts | 3–6 laps | Built into loop |
| Hill fartlek | 30–60 min | Variable | Mixed | n/a | Continuous run |

### 8.2 Short hill repeats (10–30 s)

| Field | Prescription |
|-------|--------------|
| Purpose | Power, tendon stiffness, form; gateway speed work |
| Grade | 4–7% |
| Length | 100–150 m |
| Duration | 10–30 s |
| Pace | 90–95% effort, controlled |
| Reps | 8–16 (start 8, build to 16) |
| Recovery | Walk or jog back to start; full recovery |
| Frequency | 1×/week base phase; 1× every 2 weeks specific |
| Variations | Progressive hills (build set length over weeks), hill ladder (10s-20s-30s) |

### 8.3 Medium hill repeats (60–90 s)

| Field | Prescription |
|-------|--------------|
| Purpose | Aerobic + strength stimulus; bridges short hills and long hills |
| Grade | 4–6% |
| Pace | ~5K–10K effort |
| Reps | 6–10 |
| Recovery | 2–3 min jog down |
| When in cycle | Late base, early specific |

### 8.4 Long hill repeats (3–5 min)

| Field | Prescription |
|-------|--------------|
| Purpose | VO2max with hill-strength stimulus; substitute for flat intervals when injury-prone |
| Grade | 3–5% (lower than short hills) |
| Pace | T to 10K effort. First half slightly slower, build aggressively |
| Reps | 4–8 |
| Recovery | Equal-time jog down |
| When in cycle | Specific phase; reduces orthopedic stress vs. flat intervals |

### 8.5 Hill circuit (Lydiard)

| Field | Prescription |
|-------|--------------|
| Purpose | Strength endurance bridge between base and speed phases |
| Original loop (Lydiard) | ~1.9 mi: 800m of springing/bounding uphill, 800m flat jog, 700m fast relaxed striding downhill, 800m wind sprints on bottom flat |
| Grade | ~1-in-3 ideal |
| Reps | 3–6 laps |
| Total session | 45–75 min |
| Frequency | 2–3×/week during 4-week hill phase |
| When in cycle | Lydiard hill phase: between base and anaerobic phase |
| Contraindications | High orthopedic stress; not for novice runners |
| Variations | Modern abbreviated circuit (single hill with bound up, jog flat, stride down) |

### 8.6 Hill fartlek

| Field | Prescription |
|-------|--------------|
| Purpose | Continuous-run hill stimulus; rolling-terrain race prep |
| Structure | 30–60 min run on hilly course; surge each climb, recover on descent and flats |
| Pace | Hard on uphills, easy on downhills/flats |
| Total at-pace | 5–10 min total of uphill surging |
| When in cycle | Base phase, trail/cross-country prep |
| Variations | Mona-style on hills (timed efforts), pyramid hill fartlek (escalating climb durations) |

## 9. Fartlek variations

"Fartlek" = "speed play" (Swedish). Unstructured to highly structured pace variation within a continuous run.

### 9.1 Fartlek family overview

| Workout | Total duration | Structure | Pace span |
|---------|----------------|-----------|-----------|
| Mona fartlek | 20 min | 2×90 s, 4×60 s, 4×30 s, 4×15 s; equal float | 5K → mile effort |
| Michigan fartlek | ~25–35 min + cooldown | Mile track, mile road, 1200 track, 1200 road, 800 track, 800 road, 400 track | Mile-pace (track) / steady (road) |
| Lydiard fartlek | 45–60 min | Free, by feel | Easy → hard surges |
| Time-based fartlek | 20–40 min | Pre-set work/float repeats | Variable |

### 9.2 Mona fartlek

| Field | Prescription |
|-------|--------------|
| Purpose | High-intensity total-package workout in 20 min; recovery-on-the-run skill |
| Origin | Steve Moneghetti / Chris Wardlaw, Australia |
| Structure | 2 × 90 s hard / 90 s float, then 4 × 60 s / 60 s float, then 4 × 30 s / 30 s float, then 4 × 15 s / 15 s float |
| Pace | 90 s reps at 5K effort; 15 s reps at mile effort; floats are recovery jogs (not stops) |
| Total | 20 min continuous; 14 reps |
| Warmup/cooldown | 15 min E each side |
| Frequency | Weekly or every 10 days |
| When in cycle | Base through specific; first session back from layoff (lighter version) |
| Contraindications | Can be sandbagged or overcooked easily — keep efforts honest |
| Variations | Half Mona (cut in half), Mona on hills, Mona with extended floats (more recovery) |

### 9.3 Michigan fartlek

| Field | Prescription |
|-------|--------------|
| Purpose | Cross-country race simulation: pace changes on varied surfaces |
| Origin | Ron Warhurst, University of Michigan, mid-1970s (after Steve Prefontaine's Oregon-trail variant) |
| Structure (classic) | Mile on track (~15–20 s slower than mile PR) — 3 min jog — Mile on road/trail (steady, ~min/mi slower than track mile) — 3 min jog — 1200m on track — 3 min jog — 1200m on road — 3 min jog — 800m track — 3 min jog — 800m road — 3 min jog — 400m track all-out |
| Pace span | Track segments at mile/3K effort; road segments at threshold/MP |
| Total at-pace | ~3–4 mi track + ~3 mi road |
| Frequency | 1× per training cycle (signature workout) |
| When in cycle | Peak specific phase, XC/track |
| Contraindications | Logistically demanding; substitute alternating surfaces if no road/track adjacent |
| Variations | Shortened Michigan (drop 800/400), all-track Michigan |

### 9.4 Classic Lydiard fartlek

| Field | Prescription |
|-------|--------------|
| Purpose | Free-form pace play; reintroduce surges in base phase |
| Structure | Continuous run, 45–60 min, with surges of varying length and intensity by feel |
| Pace | Easy bulk; surges from 30 s to 3 min at moderate-to-hard effort |
| Surges | 4–10 per session |
| When in cycle | Base phase; transitional weeks |
| Variations | Telephone-pole fartlek (surge between landmarks), partner fartlek (alternate who calls surges) |

### 9.5 Time-based fartlek

| Field | Prescription |
|-------|--------------|
| Purpose | Structured pace variation without track or measured course |
| Common structures | 6×3 min on / 2 min off; 8×2 min on / 1 min off; 1-2-3-4-3-2-1 min ladder |
| Pace | "On" segments at 5K–10K effort; floats are easy jog |
| Frequency | Weekly or biweekly |
| When in cycle | Base phase or trail-running prep |

## 10. Combo / alternation workouts

Continuous workouts that alternate paces without true recovery.

### 10.1 Marathon pace + 10K pace alternations

| Field | Prescription |
|-------|--------------|
| Purpose | Marathon-specific lactate-clearance training; teaches surging at race pace |
| Structure (long) | 1 mi at MP / 1 mi at 10K, repeated 5–8× |
| Structure (entry) | 1 mi MP / 400 m at 10K, repeated 6–10× |
| Total volume | 8–15 mi continuous |
| Pace | Faster segments at 10K to HM pace; recovery segments at MP (NOT easy) |
| Recovery | None — continuous |
| Frequency | Every 2–3 weeks |
| When in cycle | Marathon specific phase, 6–10 weeks out |
| Contraindications | High accumulated stress — full easy day before and after |
| Variations | Progressive (faster segments lengthen across the workout); Nate Jenkins MP/HM alternations (longer segments, less differential) |

### 10.2 Threshold + VO2 combos

| Field | Prescription |
|-------|--------------|
| Purpose | Bridge stamina and aerobic-power systems in one session |
| Common structures | 2 mi T + 4×800 I; 4 mi T + 6×400 R; 3×1 mi T + 4×1K I |
| Pace | Each block at its own zone |
| Recovery | 2–3 min between blocks; standard recoveries within each block |
| When in cycle | Late specific phase, HM/marathon |
| Contraindications | Don't combine zones if either system is undertrained — practice each in isolation first |

### 10.3 Wave tempo

| Field | Prescription |
|-------|--------------|
| Purpose | Continuous threshold with rhythmic pace variation |
| Structure | Alternate ±5–15 s/mi around T pace, 30 s to 2 min per segment |
| Total | 4–8 mi continuous |
| Pace | Average comes out near T |
| When in cycle | Specific phase HM/marathon |

## 11. Marathon-specific workouts

### 11.1 Canova special block

| Field | Prescription |
|-------|--------------|
| Purpose | Massive marathon-specific stimulus over a single calendar day; depletes glycogen and trains under-fatigue running |
| Origin | Renato Canova (elite marathon coach) |
| Structure | Two sessions, same day, ~6–8 hours apart. AM and PM both at ~90% effort. Minimal carbohydrate intake between sessions |
| Total volume | 45–50 km (28–31 mi) across the day |
| AM example | 25–30 km progressive long run, last portion at MP |
| PM example | 15–20 km with 10–12 km at MP, or 4–6 × 2K at MP |
| Frequency | 2–3× per marathon cycle |
| Spacing | 2–3 weeks between blocks |
| When in cycle | Specific phase; first block 8–10 weeks out, last 4–5 weeks out |
| Contraindications | Elite-level workout; sub-elite runners scale to ~30–40 km total. Requires high training age |
| Variations | Modified block (single longer run with two segments separated by short rest) for mortals |

### 11.2 Canova 2K repeats

| Field | Prescription |
|-------|--------------|
| Purpose | Marathon-specific aerobic; threshold-zone work in race-relevant rep length |
| Pace | Start slightly slower than MP; descend across reps to slightly faster than T (anaerobic threshold) |
| Pace progression | Each rep 2.5–5 s/km faster than the previous |
| Reps | 4–8 × 2K |
| Recovery | 2–3 min jog (60–90 s for advanced) |
| Total volume | 8–16 K at quality |
| Frequency | Every 10–14 days in specific phase |
| Pacing requirement | Even pace within each rep — Canova/Arcelli emphasize a tempo trainer |
| Variations | 2 × 3K + 3 × 2K (Kipsang pre-WR), 5 × 2K all at MP, 6 × 2K cutdown |

### 11.3 Long marathon-pace runs

See §4.4. Standard structure: easy warmup + 8–16 mi at MP. Frequency every 2–3 weeks during specific phase.

### 11.4 Pre-fatigue marathon-pace work

| Field | Prescription |
|-------|--------------|
| Purpose | Train MP execution under glycogen depletion |
| Structures | (a) 14–18 mi easy on Saturday + 6–10 mi MP on Sunday; (b) 8 mi easy + immediate 8 mi MP; (c) 2-day back-to-back doubles |
| Pace | MP exact |
| Frequency | Every 3–4 weeks specific |
| Contraindications | Highest accumulated stress workout; must be followed by ≥2 easy/recovery days |

## 12. Cutdown / progression workouts

### 12.1 Cutdown family

| Workout | Reps × distance | Pace progression | Recovery |
|---------|-----------------|------------------|----------|
| Mile cutdowns | 3–6 × 1 mi | Each rep 5–15 s/mi faster | 60–90 s jog |
| 1K cutdowns | 5–8 × 1K | Each rep 5 s/mi faster | 60–90 s jog |
| 5K progression | 5K continuous | 5K split into thirds, each faster | None |
| Continuous mile cutdown | 5–7 mi continuous | Each mile 10–15 s/mi faster | None |

### 12.2 Mile cutdowns

| Field | Prescription |
|-------|--------------|
| Purpose | Progressive load; final reps at I/R pace force composure under fatigue |
| Structure | Start slower than MP; each rep 5–15 s/mi faster. Final rep at 5K pace or faster |
| Reps | 3–6 |
| Recovery | 60–90 s jog (Daniels-style cruise rest) |
| Pace example | 6 reps: MP+10, MP, MP-10, HM, T, 10K |
| Frequency | Every 2 weeks specific phase |
| When in cycle | Specific phase, 5K/10K/HM |
| Variations | 3-2-1 mile cutdown (3 mi, 2 mi, 1 mi each progressively faster) |

### 12.3 1K cutdowns

| Field | Prescription |
|-------|--------------|
| Purpose | Smaller-dose cutdown; useful for taper sharpening |
| Structure | 5–8 × 1K, each ~5 s/mi faster |
| Pace | Start at MP, finish at 5K |
| Recovery | 60–90 s jog (or 200 m jog) |
| Frequency | Every 7–10 days specific |

### 12.4 5K progression

| Field | Prescription |
|-------|--------------|
| Purpose | Race-pace skill; teach negative split |
| Structure | 5K continuous: first third at HM, middle at T, final third at 10K-5K |
| Pace | Start ~30 s/mi slower than 5K, finish at or below 5K pace |
| Recovery | None |
| When in cycle | Specific phase 5K/10K |

### 12.5 Continuous mile cutdowns

| Field | Prescription |
|-------|--------------|
| Purpose | Tempo-cutdown hybrid; sustained progression |
| Structure | 5–7 mi continuous, each mile ~10–15 s/mi faster than prior |
| Pace | Start MP+15, drop to slightly faster than HM by final mile |
| Total | 5–7 mi at quality |
| Recovery | None |
| Frequency | Every 2 weeks specific |

## 13. Ladders

### 13.1 Ladder structures

| Type | Sequence | Target |
|------|----------|--------|
| Ascending ladder | 400-800-1200-1600 | Build mental load; finishing on long rep tests stamina |
| Descending ladder | 1600-1200-800-400 | Front-loaded; produces ~2× time at peak intensity vs equal-volume sets |
| Pyramid (up-and-down) | 400-800-1200-1600-1200-800-400 | Hits multiple zones; tests both stamina and finish |
| Compressed pyramid | 200-400-600-800-600-400-200 | 5K/mile prep |

### 13.2 400-800-1200-1600 ladder

| Field | Prescription |
|-------|--------------|
| Pace by rep length | 400 at mile/3K; 800 at 3K/5K; 1200 at 5K/10K; 1600 at 10K/HM |
| Recovery | Jog ≈ rep duration: 90 s after 400, 3 min after 800, 4 min after 1200, 4–5 min after 1600 |
| Total at-pace | 4 K (one-way); 8 K (full pyramid) |
| Warmup/cooldown | Standard track warmup |
| Frequency | Every 10–14 days specific |
| When in cycle | Specific phase XC/track |
| Variations | Reps run all at 5K (constant-pace ladder), reps each at goal pace for that distance |

### 13.3 Up-and-down ladder

| Field | Prescription |
|-------|--------------|
| Structure | 400-800-1200-1600-1200-800-400 (or 200-400-800-1200-800-400-200) |
| Recovery | Equal time jog after each rep |
| Pace | 5K to 10K range; sometimes ladder pace (faster on shorter reps) |
| Total at-pace | 8 K full version |
| Note | Descending half is mentally easier; ascending half builds toughness |

## 14. Race-specific workouts

Workouts whose paces and structures directly mirror race demands.

### 14.1 5K-specific

| Workout | Structure | Pace | Recovery |
|---------|-----------|------|----------|
| 3K reps | 2–3 × 3K | 10K to HM pace | 3 min jog |
| Mile repeats at 5K | 4–5 × 1 mi | 5K race pace | 2–3 min jog |
| 2 × (4 × 400) | Two sets, set break | 5K to 3K | 60 s within set, 3 min between sets |
| 12 × 400 at 5K | Classic 5K simulator | 5K race pace | 60–90 s jog |
| 5 × 1K at 5K pace | Threshold-volume at race pace | 5K race pace | 90 s – 2 min jog |

### 14.2 10K-specific

| Workout | Structure | Pace | Recovery |
|---------|-----------|------|----------|
| Mile repeats at 10K | 5–6 × 1 mi | 10K race pace | 60 s jog |
| 2K reps | 4–5 × 2K | 10K race pace | 2–3 min jog |
| 3 × 2 mi | Long-rep threshold | 10K to HM | 3 min jog |
| 6 × 1200 at 10K | Mid-rep volume | 10K race pace | 90 s – 2 min jog |

### 14.3 Half-specific

| Workout | Structure | Pace | Recovery |
|---------|-----------|------|----------|
| 4 × 2 mi | Predictor session | HM race pace | 60–120 s jog |
| 6 × 1 mi at HM | Mid-rep volume | HM race pace | 60 s jog |
| 3 × 3 mi at HM | Long-rep stamina | HM race pace | 3 min jog |
| 8 × 1K at HM | Sub-threshold volume | HM to T | 60 s jog |
| Long tempo | 8–12 mi continuous | HM-ish | None |

The 4 × 2 mi at HM pace with 60–120 s rest is a classic HM readiness workout. Completing it in control 2 weeks before the race indicates readiness.

### 14.4 Marathon-specific

Cross-reference §11. Primary sessions: long MP run, MP+10K alternations, Canova 2K reps, long tempos at MP/HM.

## 15. Training-cycle placement summary

| Phase | Primary workouts | Frequency |
|-------|------------------|-----------|
| Base (8–12+ wks) | E, GA, medium-long, long, strides, hill sprints, occasional fartlek/light hills | 2 quality sessions/wk max |
| Hill / strength (3–4 wks, optional) | Hill circuit, long hill repeats, hill sprints | 2–3 quality/wk |
| Specific support (4–6 wks) | T, cruise intervals, mile repeats at slower I, alternations | 2 quality/wk |
| Race-specific (4–8 wks) | Race-pace workouts, MP long runs, Canova structures, 4×2 mi for HM | 2 quality/wk + long run |
| Sharpening / taper (2–3 wks) | Reduced-volume versions of recent workouts; strides; short race-pace work | 2 lighter quality/wk |

## 16. Combinations to avoid

| Wrong combo | Why |
|-------------|-----|
| VO2max + long run within 48 hrs | Both deplete glycogen; doubles injury risk |
| MP long run + hard tempo within 5 days | Same energy system, same impact pattern, no recovery between |
| Two threshold sessions back-to-back | Only the Norwegian double-day model handles this, and only with sub-threshold pacing |
| Fast finish long run before goal race | Adds depletion in taper window |
| 400m R-pace day before threshold | Soft-tissue load incompatible with quality threshold next day |

## 17. Track session structures

### 17.1 Standard warmup protocol

| Step | Duration |
|------|----------|
| 1. Easy jog | 10–20 min (1–2 mi) |
| 2. Dynamic mobility (leg swings, hip circles, lunges) | 3–5 min |
| 3. Drills (A skip, B skip, high knees, butt kicks, carioca) | 5–8 min, ~30 m each, 2–3 passes |
| 4. Strides | 4 × 80–100 m, building, full-recovery |
| 5. Workout |  |
| 6. Cooldown jog | 10–20 min (1–2 mi) |
| 7. Optional: light static stretching | 5 min |

### 17.2 Drill sequence (canonical)

| Drill | Purpose | Reps |
|-------|---------|------|
| A march | Slow knee-drive pattern; foundation | 2 × 30 m |
| A skip | Higher-tempo knee drive; coordination | 2 × 30 m |
| B march | Add leg extension; hamstring activation | 2 × 30 m |
| B skip | High-tempo extension drill; hip mobility | 2 × 30 m |
| High knees | Quick foot turnover, core activation | 2 × 30 m |
| Butt kicks | Hamstring activation, heel recovery | 2 × 30 m |
| Carioca | Hip mobility, lateral movement | 2 × 30 m |
| Straight-leg bounding | Glute/hamstring | 1–2 × 30 m |

### 17.3 Pre-race warmup variations

| Race distance | Warmup volume | Strides | Notes |
|---------------|---------------|---------|-------|
| 5K | 2–3 mi jog + drills | 4–6 strides | Final stride 5–10 min before gun |
| 10K | 1.5–2.5 mi jog + drills | 4 strides | Slightly less than 5K |
| HM | 1–1.5 mi jog + drills | 2–4 strides | Save energy |
| Marathon | 5–15 min jog or none | 0–2 strides | Many runners walk only |
| Mile/3K | 2–3 mi + drills + 4–6 strides + 1–2 race-pace 100s | 4–6+ | Most extensive warmup |

### 17.4 Cooldown protocol

| Step | Notes |
|------|-------|
| Easy jog 10–20 min | Most important: don't stop abruptly after hard work |
| Walk 3–5 min | Bring HR down further |
| Dynamic mobility | Hip flexors, calves, hamstrings |
| Optional static stretch | Light, after the body is cool |
| Refuel within 30 min | Carbs + protein, especially after long/hard sessions |
