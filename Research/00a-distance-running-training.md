# 0A — Distance Running Training (Generic Reference)

A reference for an AI running coach. Generic across distances (5K through ultramarathon), experience levels (beginner through elite), ages, and sexes. Definitions on first use; lookup tables and formulas preferred over prose.

---

## Glossary (terms used throughout)

| Term | Definition |
|---|---|
| VO2max | Maximal rate of oxygen uptake (mL O2 / kg / min). Ceiling of aerobic power. |
| vVO2max | Running velocity that elicits VO2max. |
| LT1 | First lactate threshold; aerobic threshold. Blood lactate begins rising above resting (~2 mmol/L). Boundary of Zone 1/Zone 2. |
| LT2 | Second lactate threshold; maximal lactate steady state (~3.5–4.5 mmol/L). Highest sustainable steady-state intensity (~30–60 min). Boundary of Zone 2/Zone 3. |
| Z1/Z2/Z3 | Three-zone intensity model: Z1 = below LT1 (easy), Z2 = LT1 to LT2 (moderate/threshold), Z3 = above LT2 (hard). |
| Running economy (RE) | Submaximal oxygen cost at a given pace (mL O2 / kg / km). Lower = better. |
| Mitochondria | Organelles producing ATP via oxidative phosphorylation; density correlates with endurance. |
| Capillary density | Number of capillaries per muscle fiber; gates O2 and substrate delivery. |
| Glycogen | Stored muscle/liver carbohydrate; primary fuel above ~65% VO2max. |
| FATmax | Intensity producing the highest fat-oxidation rate (typically 60–65% VO2max in trained athletes). |
| MFO | Maximum fat oxidation rate (g/min). |
| TID | Training intensity distribution. |
| ACWR | Acute:chronic workload ratio (current 7-day load ÷ prior 28-day average). |
| RPE | Rating of perceived exertion (typically 1–10 or 6–20 Borg). |
| RED-S | Relative Energy Deficiency in Sport. |
| Super shoe | Carbon-plated, high-stack PEBA-foam racing shoe (e.g., Nike Vaporfly class). |
| Plyometric | Stretch-shortening cycle exercise (jumps, bounds). |

## Periodization

Periodization is the structured division of training into phases with progressively shifting emphasis. Four major models apply to distance running.

### Linear (traditional / Lydiard)

| Phase | Duration | Emphasis | Volume | Intensity |
|---|---|---|---|---|
| Base / aerobic conditioning | 8–16 wk | Easy mileage, long runs, strides | High, peak | Low |
| Hill / strength | 3–6 wk | Hill repeats, bounding, circuits | High | Moderate-high |
| Anaerobic / interval | 4–6 wk | VO2max, repetition (above race pace) | Moderate | High |
| Sharpening / coordination | 3–4 wk | Race-pace simulation, time trials | Moderate-low | Race-pace |
| Taper | 1–3 wk | Reduced volume, intensity preserved | Low | Race-pace touches |

Linear suits beginners and most distances. The Lydiard variant (developed by Arthur Lydiard, NZ) is the canonical example: 100% aerobic base, then hills as a transition, then track sharpening. Pure speed comes late and briefly.

### Reverse linear

Inverts the volume/intensity curve: speed and intensity early, volume later. Use cases:

| When reverse linear fits | Rationale |
|---|---|
| Half-marathon, marathon, ultra | Race-specificity is endurance-dominant; specific work belongs near race date |
| Athletes with adequate base year-round | Skips a separate "build" — extends specific endurance later |
| Athletes lacking top-end speed | Develops vVO2max early so threshold work later is at faster absolute paces |
| Off-season after track racing | Carries existing speed forward while volume rebuilds |

### Canova / Italian school

Renato Canova's framework progresses from general → fundamental → specific, where workouts increase in race-specificity over the cycle.

| Period | Length (16-wk marathon ex.) | Workout character | Long-run treatment |
|---|---|---|---|
| General | 4–6 wk | General aerobic, general speed (strides, short reps), drills | Easy long runs |
| Fundamental | 4–6 wk | Long tempos extended, then quickened; medium-long aerobic runs | Steady long runs, modest progression |
| Specific | 6–8 wk | Workouts simulate race demands at 95–105% of goal race pace | Long runs incorporate long blocks at goal pace + recovery between at 90–95% |
| Special block | every 3–4 wk through specific | Two quality sessions in one day (AM + PM), totaling 30–50 km combined | Used as a peak-stress mesocycle stimulus |

Distinguishing principle: workouts are not abandoned — they are extended in duration and quickened in pace until they resemble the race. For the marathon, specific-period long runs include 25–35 km of running between 90–105% of marathon pace.

### Block periodization (Issurin)

Concentrated 2–4-week mesocycles, each loading one or two abilities heavily, sequenced so residual effects from prior blocks support the next.

| Block | Aim | Example for distance running |
|---|---|---|
| Accumulation | Basic abilities (aerobic capacity, strength endurance) | High-volume aerobic + heavy strength |
| Transmutation | Sport-specific abilities (threshold, race-pace) | Threshold and race-specific intervals |
| Realization | Recovery + competition prep | Reduced volume, race-pace touches, taper |

Meta-analysis evidence: block periodization produces small-to-moderate superior effects on VO2max and Wmax versus traditional periodization, particularly in already-trained athletes. Best fit: athletes with multiple races per season, advanced trainees who plateau on linear plans.

### Periodization choice by athlete and event

| Athlete profile | Default periodization |
|---|---|
| Beginner, any distance | Linear |
| Marathon, intermediate-advanced | Linear or Canova |
| Half-marathon, intermediate-advanced | Linear or reverse linear |
| 5K/10K track-trained | Linear (Lydiard) or block |
| Ultra | Linear; specific period emphasizes back-to-back long efforts |
| Year-round racer (multiple peaks) | Block |
| Returning from layoff | Linear (rebuild base before any sharpening) |

## Training Intensity Distribution (TID)

TID is the proportion of training time (or sessions) spent in Z1, Z2, Z3.

| Distribution | Z1 (easy) | Z2 (threshold) | Z3 (hard) | Hallmarks |
|---|---|---|---|---|
| Polarized | ~80% | ~5% | ~15% | Mostly easy; intense work above LT2 |
| Pyramidal | ~80% | ~15% | ~5% | Easy base; meaningful threshold; little above LT2 |
| Threshold-dominant | ~50–70% | 25–45% | <10% | Heavy threshold load (e.g., classic Norwegian model) |
| HVLIT (high-volume low-intensity) | >90% | <10% | minimal | Pure base building |

### When each TID applies

| Race distance | Phase | Best-fit TID | Rationale |
|---|---|---|---|
| 5K/10K | Base | Pyramidal or polarized | Build aerobic capacity broadly |
| 5K/10K | Specific | Polarized | Z3 demands of race require above-LT2 stimulus |
| Half-marathon | Base | Pyramidal | LT2 and economy dominate the event |
| Half-marathon | Specific | Pyramidal or threshold-dominant | Most race time is at/just below LT2 |
| Marathon | Base | Pyramidal | LT2 and fatigue resistance dominate |
| Marathon | Specific | Pyramidal | Race intensity sits at 80–88% LT2 — Z2 work is race-specific |
| Ultra | Base / specific | HVLIT / pyramidal | Aerobic capacity, fat oxidation, fatigue resistance |

### Norwegian double threshold

Two sub-LT2 ("controlled threshold") sessions in one day, 2 days/wk, both kept at 2.0–3.5 mmol/L lactate. Origins: Marius Bakken; popularized via the Ingebrigtsen brothers; for the Ingebrigtsens 23–25% of weekly mileage in prep ran at/near anaerobic threshold.

| Element | Specification |
|---|---|
| Frequency | 2 double-threshold days/wk (e.g., Tue + Thu) |
| AM session | Longer reps (1–2 km) at 2.5–3.0 mmol/L |
| PM session | Shorter reps (200–600 m at threshold pace) at 2.5–3.5 mmol/L |
| Other days | Easy aerobic, one long run, optional long intervals or strides |
| Lactate cap | Stay below LT2 (typically <4.0 mmol/L) — never accumulate |
| Required infrastructure | Lactate meter or close pace-discipline |
| Suitable for | Athletes running ≥80–100 km/wk who can fit double sessions |

The mechanism: by capping intensity below LT2, total threshold volume can be far higher than from one all-out tempo, with smaller systemic disruption.

### Norwegian singles

A single sub-threshold workout every other day, for athletes who cannot do doubles. Total threshold mileage 20–25% of weekly volume. Suits 5K-to-marathon trainees with established 80–120+ km/wk.

| Norwegian singles week (sample structure) | |
|---|---|
| Day 1 | Sub-threshold session (e.g., 5×1 km at LT2 pace + 5–10 sec) |
| Day 2 | Easy + strides |
| Day 3 | Sub-threshold session (e.g., 8–10×400 m at LT2 pace) |
| Day 4 | Easy |
| Day 5 | Sub-threshold session (e.g., 4×2 km) |
| Day 6 | Easy or off |
| Day 7 | Long run (mostly easy, optional progression at end) |

### TID — the disagreement and when each TID matters

Stöggl & Sperlich (2014): polarized produced largest endurance gains across well-trained athletes vs. HVLIT, threshold-dominant, and HIIT. But subsequent run-specific work showed mixed results: some athletes respond better to pyramidal (especially for marathon), and individual responder profiles cluster (~30% polarized responders, ~30% pyramidal responders, ~20% dual responders, ~20% non-responders). Practical synthesis:

- All elite distance runners — regardless of system — converge on ≥75% of training volume in Z1.
- Differences live in the remaining 20–25%: how much Z2 vs. Z3.
- Race-distance specificity tilts the answer: shorter events (1500–10K) reward more Z3; marathon and longer reward more Z2.

## Aerobic Base Development

The base period builds the aerobic engine. The targets:

| Adaptation | What it does | Primary stimulus | Time course |
|---|---|---|---|
| Mitochondrial biogenesis | More ATP from fat and carbohydrate | Easy/moderate volume; PGC-1α activation by AMPK and Ca²⁺ signaling | Weeks; volume-dependent |
| Capillary density | O2 + substrate delivery; CO2 + lactate clearance | Volume across all intensities; gains saturate after several weeks in trained | Untrained: <4 wk for first gains; trained: continuous slow progress |
| Type I (slow-twitch) hypertrophy / oxidative shift | Higher fatigue resistance at submax | Long aerobic running | Months |
| Hybrid → Type IIa shift | More fatigue-resistant fast fibers | Threshold / VO2max work | Weeks–months |
| Plasma volume expansion | Stroke volume, thermoregulation | Volume + heat | Days–weeks |
| Stroke volume / cardiac output | Higher VO2max | High-volume + intervals | Months |
| Fat oxidation enzymes (CPT-1, β-HAD) | Higher MFO; glycogen sparing | Long, easy runs (especially fasted) | Weeks |
| Tendon / bone remodeling | Injury resistance | Cumulative loading; gradual progression | Months–years |

PGC-1α is the master regulator coordinating mitochondrial biogenesis, capillary growth (via VEGF), and substrate metabolism. It is activated by repeated bouts of muscle contraction; both volume and intensity contribute, but volume is the more consistent route to mitochondrial content gains.

Capillary growth in untrained-to-moderately-trained athletes occurs within the first 4 weeks. Trained athletes accrue capillary density gains slowly with continued volume. Type I fibers possess: rich capillary supply, high mitochondrial density, abundant myoglobin, high oxidative enzyme activity.

### Practical base-building rules

| Rule | Application |
|---|---|
| Most base running is easy | 75–90% in Z1 |
| Long run grows | Up to 25–30% of weekly volume (cap before injury risk) |
| One Z2 stimulus minimum | Even base periods benefit from a single tempo or steady-state per week |
| Strides preserved | 4–8×100 m strides 1–2×/wk maintain neuromuscular function |
| No racing | Hard racing during base undermines the aerobic emphasis |
| Hill strides | Optional — develop running-specific strength |

## Volume Guidelines by Experience and Distance

Volumes are weekly running mileage at peak training (excluding taper). Assume year-round running.

### Volume table — miles per week (km in parentheses)

| Distance | Beginner (just finishing) | Recreational competitive | Sub-elite | Elite |
|---|---|---|---|---|
| 5K | 10–20 (16–32) | 25–40 (40–65) | 50–70 (80–115) | 70–110 (115–175) |
| 10K | 15–25 (24–40) | 30–50 (50–80) | 55–80 (90–130) | 90–130 (145–210) |
| Half-marathon | 20–30 (32–50) | 35–55 (55–90) | 65–90 (105–145) | 100–140 (160–225) |
| Marathon | 25–40 (40–65) | 40–60 (65–95) | 70–110 (115–175) | 120–160 (195–260) |
| 50K | 30–45 (50–70) | 50–70 (80–115) | 70–110 (115–175) | 120–160 (195–260) |
| 100K | 40–60 (65–95) | 55–80 (90–130) | 80–120 (130–195) | 130–180 (210–290) |
| 100-mile | 45–65 (70–105) | 60–90 (95–145) | 90–130 (145–210) | 130–200 (210–320) |

Above ~50–70 mi/wk, marginal returns diminish. The first 30 mi/wk produces the largest improvements; each additional 10 mi/wk above 70 contributes less per mile.

### Volume progression rules

| Rule | Specification |
|---|---|
| Year-on-year base growth | 5–15% per training cycle for trained athletes; novices safely +20–25% over 8 weeks vs. +10% over 12 in trial data |
| Down weeks | Every 3–4 wk, reduce by 20–30% |
| Single-session spike threshold | An individual run >110% of longest run in the prior 30 d raises overuse injury risk by ~64%; >130% raises it further |
| Long-run cap | ≤25–30% of weekly volume (or by absolute time: <3.0–3.5 h for marathoners; ultra athletes go longer) |
| Frequency | 4–7 days/wk; doubles add ~20–40% volume without per-session stress increase |
| Cross-train substitution | Bike or pool load ≈ 50–70% of run-equivalent stress on tissues |

## The Seven Workout Categories

Every distance runner's session belongs to one of these categories. Each has a primary purpose, dose, and pace target.

### 1. Recovery run

| Element | Specification |
|---|---|
| Purpose | Active recovery, blood flow, easy aerobic stimulus, frequency |
| Pace | 60–70% HRmax; conversational; ~75–90% of typical easy pace |
| Duration | 20–45 min |
| RPE | 2–3 / 10 |
| Frequency | After hard sessions; one of multiple easy days |

### 2. General aerobic (easy run)

| Element | Specification |
|---|---|
| Purpose | Mitochondrial volume, capillarization, fat oxidation; staple of weekly volume |
| Pace | 65–75% HRmax; ~80–90% LT1 pace; 60–90 sec/km slower than marathon pace |
| Duration | 40–75 min |
| RPE | 3–4 / 10 |
| Frequency | 2–4×/wk; bulk of weekly Z1 |

### 3. Medium-long run

| Element | Specification |
|---|---|
| Purpose | Aerobic strength under fatigue without long-run cost |
| Pace | Easy to steady; latter portion may drift to upper-Z1/lower-Z2 |
| Duration | 75–110 min (12–20 km / 8–14 mi typical) |
| RPE | 4–5 / 10 |
| Frequency | 1×/wk in marathon and half cycles; optional in 5K/10K |

### 4. Long run

| Element | Specification |
|---|---|
| Purpose | Fatigue resistance, glycogen depletion adaptation, fat oxidation, mental durability |
| Pace | Easy unless prescribed otherwise (see variations below) |
| Duration | 90 min – 4 h depending on race distance and athlete |
| RPE | 4–6 / 10 |
| Frequency | 1×/wk |

### 5. Threshold / tempo

| Element | Specification |
|---|---|
| Purpose | Raise LT2; expand "comfortably hard" capacity |
| Pace | LT2 pace ≈ ~83–88% VO2max ≈ 88–92% HRmax ≈ ~1-hour race pace |
| Continuous form | 20–40 min sustained |
| Cruise-interval form | 3–5×5–10 min @ LT2 with 60–90 sec jog |
| RPE | 6–7 / 10 |
| Frequency | 1–2×/wk |
| Total Z2 dose | 20–60 min/session; up to 10–12 km in advanced |

### 6. VO2max

| Element | Specification |
|---|---|
| Purpose | Raise VO2max ceiling; vVO2max; aerobic power |
| Pace | 95–100% vVO2max ≈ 3K–5K race pace |
| Interval length | 2–6 min (typically 800–1600 m or 3–5 min) |
| Recovery | ~50–100% of work duration, jogging |
| Total work | 10–20 min above 90% VO2max — roughly 15–30 min of total intervals |
| RPE | 8–9 / 10 |
| Frequency | 1×/wk during specific phase (less frequent during base) |

Long intervals (3–5 min) accrue more time at >90% VO2max than short, intensified intervals — a determining variable for VO2max gains.

### 7. Race-specific

| Element | Specification |
|---|---|
| Purpose | Adapt to demands of target race pace and distance |
| Pace | Goal race pace ± a small margin |
| Structure | Workouts mimic race effort, often as long blocks (e.g., marathon: 25–35 km with 60–80% at goal pace) |
| RPE | 5–8 / 10 (varies by race distance) |
| Frequency | 1×/wk in specific phase |

### Workout dose by race distance

| Race | VO2max | Threshold | Race-specific block |
|---|---|---|---|
| 5K | 4–8×800–1000 m at 5K pace, jog rec | 3–4×8 min cruise OR 20-min steady tempo | 5–6×1000 m at goal 5K pace, short rec |
| 10K | 5–6×1000–1200 m at 10K pace | 4×10 min OR 30–40 min steady | 3–4×2 km at goal 10K pace |
| Half-marathon | 4–5×1000 m at 5K-10K pace | 5–6×1 mi at half-marathon pace OR 8–12 km steady | 12–16 km with 8–10 km at goal half pace |
| Marathon | 4×1600 m at 5K pace (sparingly) | 8–14 km at LT2 pace; or 2×5 km with 3 min jog | 25–35 km long run with 16–25 km at goal marathon pace |
| 50K | 3×1600 m at 10K pace (rarely) | 6–10 km steady tempo | 3–5 h long run; back-to-back long days |
| 100K / 100 mi | 2–3×1600 m at 5K-10K pace (rarely) | 30–45 min tempo | 4–6 h long runs, vert-specific where applicable; back-to-back weekend doubles |

## Long-Run Variations

Long runs are not monolithic. Variants apply across distances.

| Variant | Structure | Best fit | Caution |
|---|---|---|---|
| Steady (easy) | All at easy pace | Base building, all distances; recovery weeks; first long runs of cycle | None — the default |
| Progression | Start easy; finish at marathon pace or faster | Marathon/half/10K specific phases | Don't make every long run a progression — rotate |
| Marathon-pace (MP) | Prescribed segment(s) at goal MP within a long run; e.g., 32 km with 24 km at MP | Marathon specific phase | Skip if accumulated fatigue is high |
| Fast finish | Final 10–25% at half-marathon-to-MP effort | Builds late-race resilience | Dial back if injury-prone |
| Dress rehearsal | Mid-cycle long run that simulates race day: pre-race meal, race kit, planned fueling, race shoes if applicable | Marathon, half, ultra | Once per cycle, 3–6 wk before race |
| Back-to-back long | Long day followed by another long day | Ultra prep | Cumulative load — only when base supports |
| Easy long with surges | Easy run with 6–12×30–60 sec strides or 2–4 min moderate insertions | Base period across distances | Not a hard workout |
| Hilly long | Easy effort over hills; or hilly long with surges | Trail/hilly course prep | Eccentric load — recovery may take longer |

### Long-run rules of thumb

- Long run cap: 25–30% of weekly volume.
- Time on feet (especially for ultra) is more important than distance.
- Most long runs are easy; intensity inserts come 1 in every 2–3 long runs in marathon/half cycles.
- Marathon-pace long runs of 25–35 km with 60–80% at goal pace are signature workouts in the Canova/Pfitzinger schools.

## Strength Training for Runners

Heavy resistance training and plyometrics improve running economy and time-trial performance. Both work; heavy resistance has the larger effect.

### Heavy resistance vs. plyometrics — the evidence

| Modality | Effect on running economy (Hedges' g) | Effect on time-trial performance | Notes |
|---|---|---|---|
| Heavy resistance | g ≈ −0.32 (small, favors HR) | Larger effect than plyometrics | ≥90% 1RM produced larger effects than lighter loads |
| Plyometrics | Smaller effect on RE | Smaller effect on TT | Stretch-shortening adaptations |
| Combined | Often used in practice | Plausibly additive | Most training programs include both |

### Periodization opposing the run cycle

Strength is loaded heavily when running intensity is lowest, and tapered as race-specific run intensity climbs.

| Run phase | Strength volume | Strength intensity | Exercise selection |
|---|---|---|---|
| Off-season | Build | Moderate | Foundational compound + accessory + plyo |
| Base | High | Heavy (≥85% 1RM) | Squat, deadlift, hip thrust, calf raise, single-leg work, plyo |
| Specific | Reduced volume; maintain intensity | Heavy | 1–2 exercises, 2–4 sets at 85–90% 1RM; low-volume plyo |
| Taper | Maintenance | Low volume; moderate | Light maintenance lift 7–10 d out, none in last 5 d |
| Race week | None heavy | Bodyweight only if at all | — |

### Exercise selection (generic template)

| Category | Examples | Sets×reps |
|---|---|---|
| Bilateral compound | Back squat, trap-bar deadlift | 3–5×3–6 at 85–95% 1RM |
| Unilateral lower | Rear-foot elevated split squat, single-leg RDL | 3×6–8 |
| Posterior chain | Hip thrust, RDL, hamstring curl | 3×6–10 |
| Calf | Heavy calf raise (straight + bent knee) | 3×6–10 |
| Plyometric (low) | Pogo hops, ankle bounces, A-skips | 3×10–20 contacts |
| Plyometric (high) | Box jumps, depth jumps, single-leg bounds | 3×4–6 |
| Core | Plank variations, dead bug, Pallof, anti-rotation | 2–3 sets |

### Frequency

| Athlete level | Days/wk during base | Days/wk during specific |
|---|---|---|
| Beginner | 2 | 1 (maintenance) |
| Recreational competitive | 2–3 | 1–2 |
| Sub-elite / elite | 2–3 | 1–2 |

8–12 wk programs of low-to-high intensity resistance + plyometrics 2–3×/wk reliably improve RE in trained middle/long-distance runners.

## Fueling

### During-race carbohydrate intake by race duration

| Race duration | Carb target | Carb type / ratio |
|---|---|---|
| <45 min (5K, often 10K) | None required; mouth-rinse possible | — |
| 45–75 min (10K, fast HM) | 30 g/h optional | Glucose / maltodextrin |
| 75–150 min (HM, slow HM, fast marathon) | 30–60 g/h | Glucose-only or 2:1 glucose:fructose |
| 2–3 h (most marathons) | 60–90 g/h | 2:1 glucose:fructose |
| >3 h (slow marathon, ultra) | 60–120 g/h | 1:0.8 glucose:fructose for >90 g/h |
| Ultra (>4 h) | 60–120 g/h carbs + savory + fat (~250–333 kcal/h) | Multi-source; real food at aid stations |

Mechanism: glucose absorbs via SGLT1 (saturates ~60 g/h); fructose absorbs via GLUT5 (independent transporter). Combining the two raises total absorption beyond the single-transporter ceiling.

| Glucose:fructose ratio | Best for |
|---|---|
| 2:1 | ≤90 g/h carb intake |
| 1:0.8 | >90 g/h carb intake |
| 1:1 | High-rate fueling (>100 g/h) — practiced in elite ultra-endurance |

### Daily training nutrition

| Nutrient | Target | Notes |
|---|---|---|
| Carbohydrate | 5–7 g/kg/d (moderate days) | 7–10 g/kg/d on heavy/long days |
| Protein | 1.4–1.7 g/kg/d | Spread across 4–5 feedings (~0.3 g/kg per feeding); higher (up to 2.0) for masters or during deficit |
| Fat | 20–35% energy | Don't crowd out carbs on hard-training days |
| Energy availability | ≥45 kcal/kg FFM/d | <30 kcal/kg FFM/d → RED-S risk |
| Iron (especially female) | RDA + monitor ferritin | Ferritin >35 ng/mL preferred for runners |
| Vitamin D | Maintain >30 ng/mL (75 nmol/L) 25-OH-D | Test annually |

### Post-run recovery

| Window | Target |
|---|---|
| 0–60 min | 1.0–1.2 g carb/kg + 0.25–0.30 g protein/kg |
| Through 4 h | 1.0–1.2 g carb/kg/h to refill glycogen |
| 24 h | Total daily targets met |

Carb:protein ratio ~3–4:1 in the immediate window; protein form is less important than total dose if eaten within hours of a workout.

### Carbohydrate loading

For races >90 min, carb-load 36–48 h pre-race.

| Element | Specification |
|---|---|
| Daily intake | 10–12 g carb/kg/d for 36–48 h |
| Composition | Predominantly low-fiber, easy-digesting starches/sugars |
| Hydration | Increased fluid; ~3 g water stored per 1 g glycogen |
| Weight gain | +1–2 kg expected (water + glycogen) — normal |
| Glycogen elevation | 50–100% above resting baseline |
| Race-morning meal | 1–4 g carb/kg, 2–4 h pre-start |

For shorter races (<90 min), exhaustion of glycogen is unlikely; standard dietary carb intake suffices.

## Recovery Modalities Ranked by Evidence

| Modality | Evidence strength | Mechanism | Effective dose |
|---|---|---|---|
| Sleep | Highest | Growth hormone secretion (70–80% in slow-wave sleep), tissue repair, glycogen restoration, immune function | 7–9 h adults; elites benefit from 9–10 h; nap option 20–90 min |
| Adequate fueling/hydration | Highest | Glycogen and protein substrate; plasma volume | Match daily targets above |
| Cold-water immersion (CWI) | Moderate–high (DOMS, perceived) | Vasoconstriction; reduced swelling; perceptual recovery | 10–15 min at 5–15°C; useful between same-day or back-to-back sessions |
| Compression garments | Moderate | Reduced edema; venous return; perceptual benefit | Worn 12–48 h post-session |
| Massage | Mixed (perceived ↑, performance =) | Vasodilation; perceptual recovery | 20+ min sessions; not a substitute for sleep/nutrition |
| Active recovery (easy spin/jog) | Moderate | Lactate clearance (acutely); blood flow | 15–30 min easy |
| Pneumatic compression (boots) | Moderate (perceived) | Venous return | 20–60 min |
| Foam rolling / SMR | Low–moderate (mobility ↑, performance =) | Soft-tissue mobility; perceptual | 5–15 min |
| Sauna | Moderate (heat acclimation effect, plasma volume) | Heat stress | 20–30 min post-run, 3–5×/wk |
| Stretching | Low | No reliable performance effect | Not a recovery requirement |
| Cryotherapy chamber | Low (vs. CWI: similar–weaker) | Cold stress | 2–4 min |
| NSAIDs for routine recovery | Avoid | Blunts adaptive signaling; GI risk | Reserve for medical use |

Notes:
- CWI may blunt some adaptive signaling (e.g., hypertrophy response); useful for back-to-back competition or congested race blocks; avoid routinely after key adaptive sessions.
- Combined modalities (e.g., active + massage) outperform single modalities for short-term restoration.

## Footwear Strategy

### Super shoes (carbon-plated, high-stack PEBA-foam)

| Element | Specification |
|---|---|
| Running economy improvement | ~4–6% on average (range 2–8% individual) |
| Marathon time saving | ~2–3 min for sub-elite; ~6–9 min for 4-h marathoner |
| Mechanism | Energy return from PEBA foam; carbon plate stiffens MTP joint and reduces metabolic cost; high stack offsets damping cost |
| Best for | Race day, high-stress key workouts (marathon-pace blocks, race-specific sessions) |
| Limitations | Reduced foot/ankle muscle work risks deconditioning if used as daily trainer; durability typically 200–400 km |

### Rotation

A 22-week prospective study found runners using ≥2 shoe models had ~39% lower injury risk vs. single-shoe users.

| Shoes per rotation | Application |
|---|---|
| 1 | Adequate for 1–3 runs/wk |
| 2 | Recommended baseline for 3+ runs/wk |
| 3–4 | Optimal for 4+ runs/wk; one daily trainer + one tempo/uptempo + one super shoe + (optional) trail or recovery |
| 5+ | Diminishing returns |

### Training rotation template (5+ runs/wk)

| Slot | Shoe character | Use |
|---|---|---|
| Daily trainer | Cushioned, durable, neutral foam | Easy + general aerobic + recovery |
| Uptempo / lightweight trainer | Lighter, responsive foam, sometimes plated | Tempo, threshold, MP work |
| Super shoe | Carbon plate + high-stack PEBA | Race-specific workouts and races |
| Trail / off-road | Aggressive lugs, rock plate | Trail or mixed surfaces |

### Replacement and use

| Shoe class | Typical lifespan |
|---|---|
| Daily trainer | 500–800 km |
| Tempo trainer | 400–600 km |
| Super shoe | 200–400 km |
| Trail shoe | 500–800 km (outsole) |

## Cadence and Form

### Cadence (steps per minute)

| Range | Description |
|---|---|
| 150–170 SPM | Lower than typical recreational; associated with higher impact and overuse injury risk in some studies |
| 170–180 SPM | Common range for recreational runners |
| >180 SPM | Common for elite runners |
| Self-selected | Generally efficient — large arbitrary changes may reduce economy |

### Cadence rules

- Cadence is speed-dependent: it rises naturally with pace.
- Increasing cadence by 5–10% above self-selected cadence reduces vertical ground reaction force, loading rate, vertical oscillation, and improves hip/knee alignment without compromising economy in most runners.
- Arbitrary "180 SPM" targets are not evidence-based for all body sizes. Taller athletes typically run at lower cadences at the same pace.
- Use cadence cues conservatively (5% bumps); change accumulates over weeks.

### Form (general principles)

- Forefoot vs. heel strike: footstrike pattern alone does not predict economy or injury. The same runner may strike differently across paces.
- Posture: tall, stacked alignment (ear-shoulder-hip-ankle); avoid excessive forward lean from waist.
- Arm carriage: relaxed, swing fore-aft (not across body).
- Drills (A-skips, B-skips, butt kicks, strides): improve neuromuscular coordination — useful 1–2×/wk regardless of distance.

## Heat Acclimation

Most adaptations complete within 10–14 days of daily exposure.

| Adaptation | Magnitude | Time |
|---|---|---|
| Plasma volume expansion | +6–12% | 5–7 d |
| Sweat rate ↑ | Substantial | 4–10 d |
| Sweat onset earlier | — | 4–8 d |
| Sweat sodium ↓ | — | 7–14 d |
| Resting core temp ↓ | ~0.2°C | 7–14 d |
| Heart rate at given pace ↓ | 10–15 bpm | 5–10 d |
| Performance in heat | +6–8% | 10–14 d |
| Performance in cool conditions | Small/variable | — |

### Protocol

| Element | Specification |
|---|---|
| Mode | Run, indoor sauna, hot tub, hot bath, overdressed running |
| Core temp target | ≥38.5°C |
| Duration | 60 min/session (continuous or split) |
| Frequency | Daily for 10–14 days |
| 7-day option | ~70–80% of full benefit |
| Maintenance | 1–2 sessions/wk preserves adaptations |
| Decay | Plasma volume reverts in 1–2 wk without exposure |

Sauna acclimation works at lower thermal stress: 20–30 min at ~80–90°C post-run, 4–5×/wk for 2–4 wk.

## Altitude

### Live-high, train-low (LHTL)

| Element | Specification |
|---|---|
| Living altitude | 2000–2500 m |
| Daily exposure | ≥20 h/d |
| Duration | 3–4 wk |
| Training altitude | 0–1200 m (sea level) when possible |
| Hb mass increase | ~3–7% |
| Performance gain (sea-level race) | ~1–4% in responders; non-trivial proportion are non-responders |
| Marathon model estimate | ~8–9 min improvement in well-modeled cases |

### Live-high, train-high (LHTH)

Lower training quality at altitude offsets some hematological benefit. Used historically; LHTL is preferred when feasible.

### Practical points

| Issue | Note |
|---|---|
| Iron status | Test ferritin pre-camp; supplement if low — Hb mass cannot rise without iron |
| Re-acclimation | First 2–3 d at altitude: easy only; intensity returns after 4–7 d |
| Race timing post-camp | 2–4 wk post-descent is a common peak window |
| Variability | Substantial individual response; some athletes are non-responders |
| 5K/10K vs. marathon | All endurance distances benefit; the more aerobic the event, the larger the expected gain |

## Age-Related Training Adjustments

### By stage

| Stage | Ages | Key adjustments |
|---|---|---|
| Youth | <14 | Multi-sport, play-based; running is one of many activities; avoid high specialization |
| Adolescent | 14–18 | Gradual specialization; cap weekly mileage; monitor growth-plate injuries; lower-mileage, fundamentals-first approach |
| Young adult | 19–34 | Peak training capacity in most |
| Masters early | 35–50 | Maintain volume; introduce strength training if not already; recovery days emphasized |
| Masters mid | 50–65 | Reduce frequency of hard sessions; preserve intensity; strength work is non-negotiable |
| Masters late | 65+ | Low-mileage, higher-quality approach often productive; prioritize bone and balance work |

### Youth and adolescent specifics

- Bone stress injury incidence in adolescent athletes: 3.9–19%, with up to 21% recurrence. Tibia, fibula, metatarsals are common sites.
- Female adolescent runners show higher injury rates than males (56% vs. 50%) and higher BSI rates (6.7% vs. 3.8%).
- Early single-sport specialization is associated with elevated injury risk.
- Cap weekly running volume and limit consecutive run days during rapid growth phases.
- Skip race specialization; 800–5K range at this age is plenty of distance.

### Masters: VO2max decline

- VO2max declines ~5% per decade in trained masters; up to ~10% per decade if training drops.
- ~50% of decline tracks training volume reductions.
- Maximal heart rate and stroke volume drop with age; running economy is largely preserved if training continues.
- Type I fiber proportion is largely preserved with continued training.

### Masters: training adjustments

| Adjustment | Rationale |
|---|---|
| Maintain volume — don't drop first | Volume preserves VO2max more than intensity does |
| Strength training 2–3×/wk | Counters sarcopenia, preserves running economy and bone |
| Add a recovery day | Recovery time lengthens with age |
| Reduce hard-session frequency from 2–3/wk → 2/wk | Restoration of high-intensity capacity is slower |
| Increase warm-up | 15–25 min easy + drills before quality |
| Prioritize sleep, hydration, protein | Adaptations narrower margin |
| Cross-train | Bike, pool, elliptical reduce impact stress |
| Plyometrics | Preserve power output; low-volume contacts |

## Sex-Specific Training Considerations

### Menstrual cycle

Standard cycle phases:

| Phase | Days (28-d cycle) | Hormonal profile |
|---|---|---|
| Early follicular (menses) | 1–5 | Low estrogen, low progesterone |
| Late follicular | 6–14 | Rising estrogen, low progesterone |
| Ovulatory | ~14 | Estrogen peaks |
| Mid/late luteal | 15–28 | Both estrogen and progesterone elevated |

### Menstrual cycle and performance — what the evidence shows

- Most well-controlled studies find no consistent group-level differences in endurance time-trial performance across menstrual cycle phases.
- Some individual athletes report consistent symptom-driven performance shifts; these are real and should be tracked individually.
- Plasma volume drops ~8% in mid/late luteal — hydration matters more here, especially in heat.
- Estrogen promotes fat oxidation; progesterone partially counters this. Net effect on endurance is small.
- Strength training adaptations: current best evidence shows no advantage to phase-based periodization vs. standard programming.

### Pragmatic guidance

| Phase | Default training | Symptom-management |
|---|---|---|
| Early follicular | Normal training; some athletes feel best | Iron-rich foods if heavy menses |
| Late follicular | Higher-intensity work tolerates well | — |
| Mid-luteal | Normal training; expect higher RPE for given pace | Hydration + electrolytes; longer warm-up |
| Late luteal/PMS | Normal training; honor individual symptoms | Anti-inflammatory foods; flexibility on hard days |

Track cycles ≥3 months before drawing patterns. Avoid making cycle-driven adjustments based on a single experience.

### Hormonal contraception

| Type | Endurance effect (group level) |
|---|---|
| Combined oral contraceptive | ~5% lower VO2max in some studies; minimal effect on actual endurance performance |
| Progestin-only / IUD | Low / inconsistent group-level effects |
| Withdrawal week (placebo pills) | RPE may shift; performance generally stable |

Choice of contraception should be a medical and personal decision; performance differences are typically small at the group level.

### RED-S (Relative Energy Deficiency in Sport)

Energy availability (EA) = (energy intake − energy cost of exercise) ÷ fat-free mass.

| EA (kcal/kg FFM/d) | Status |
|---|---|
| ≥45 | Adequate |
| 30–44 | Subclinical risk |
| <30 | High-risk LEA / RED-S |

Signs to monitor (any sex):

| Signal | Implication |
|---|---|
| Menstrual irregularity / amenorrhea | LEA marker (when applicable) |
| Decreased libido | Lowered sex-steroid signaling |
| Recurrent stress fractures | Bone-mineral consequence |
| Cold intolerance, hair thinning | Thyroid suppression |
| Persistent fatigue, declining performance despite training | Endocrine and metabolic consequences |
| Frequent illness | Immune suppression |
| Mood changes, depression | Neuropsychological consequences |
| GI complaints | LEA-associated gastric motility changes |
| Resting HR drop + increased orthostasis | Cardiovascular adaptation to LEA |

LEAF-Q and similar screening tools formalize this assessment. Both sexes are at risk; symptoms are not female-only.

Action: refer for clinical evaluation; correct EA before chasing further training adaptations.

## Training Load and Injury Risk

### Load metrics

| Metric | Calculation |
|---|---|
| Acute load (7-day) | Sum of session load (RPE × duration, or distance, or weighted by intensity) over last 7 days |
| Chronic load (28-day) | Mean weekly load over last 28 days |
| ACWR | Acute ÷ Chronic |
| EWMA-ACWR | Same ratio with exponentially weighted recent loads (more sensitive at high ACWR) |

### ACWR risk zones

| ACWR | Status | Injury risk |
|---|---|---|
| <0.8 | Detraining / low load | Slight increase (under-prepared for return) |
| 0.8–1.3 | Sweet spot | Lowest risk |
| 1.3–1.5 | Caution | Moderate |
| ≥1.5 | High | Substantially elevated |

ACWR is a heuristic, not a rule. It does not capture session character, surface, or previous injury. Recent meta-analyses show variable predictive power; use it as a load monitor rather than a forecast.

### The 10% rule — reconsidered

The traditional "increase weekly mileage by ≤10%" rule is not strongly supported by recent evidence:

| Finding | Source |
|---|---|
| Novice runners progressing weekly volume +24% over 8 wk did not show higher injury rates than +10% over 12 wk | RCT (Buist et al.) |
| Single-run length spike >110% of longest run in prior 30 d → 64% increased overuse injury risk | BJSM 5,200-runner cohort |
| Weekly mileage change correlated weakly with injury | Same cohort |

### Practical load rules

| Rule | Specification |
|---|---|
| Long-run cap rule | Single long run should not exceed 110% of the longest run in the prior 30 days |
| Weekly down-weeks | Reduce 20–30% every 3–4 weeks |
| Hard-session spacing | 48 h between hard sessions |
| Add stress one-at-a-time | Either add mileage OR add intensity in a given week, not both |
| Surface variation | Mix surfaces; don't add hill volume + speed at once |
| Fatigue gate | Honest RPE on easy runs, persistent elevated resting HR, sleep disturbance → reduce |
| Past injury | Strongest predictor of next injury — adapt plan to recurrence patterns |
