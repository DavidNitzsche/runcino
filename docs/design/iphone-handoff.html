Per-screen build spec

Job · data source · key components · primary interaction. The agent builds against this.
ScreenJobData sourceKey components / interaction

Workout detailBrief + push to watchPlan workout + targetPaces from VDOTStructure rows, WHY block, fueling. Primary: Send-to-Watch. Secondary: move/swap/skip sheet.
Run recapHow did it goHealthKit workout ← watch · reconcileActivity()Auto-modal on app open after a run. Hero stats, reps vs target, coach read, RPE capture, share.
PlanSee + lightly editPlan phases + per-day workoutsWeek list, day rows, done/today markers. Tap day → detail. Move/swap/skip; deep edit → web.

Race day modeGuide me, racing nowLive run + race plan + fuelingPace vs goal (color), splits, fuel cues, projection. Watch primary; phone secondary. Auto-recap at finish.
SettingsManage accountIntegration + permission stateProfile, integrations + sync status, notification prefs, units, watch management, card visibility.
Live ActivityGlance without openingIn-run feed · race countdownActivityKit. Lock screen + Dynamic Island. Live pace/HR/rep during runs.
