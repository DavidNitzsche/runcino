How to pick
Each direction trades a different way:
  • A · Story Hero — narrative-first, most "Faff voice." Best if you want the run to feel coached, not measured. Trade: hides raw splits behind the checklist.
  • B · Coach's Sheet — info-dense, single card, table-based. Best if you want everything visible at once with zero scrolling games. Trade: less emotional, more spreadsheet.
  • C · Scoreboard — glanceable, pass/fail dots. Best if you want a 1-second read of "how did it go." Trade: less context per metric.
  • D · Magazine — editorial, pull-quote, two columns. Best if the run-detail moment should feel like reading about your run. Trade: takes more vertical space than B or C.

Pick one (or a hybrid — "A's hero with C's scorecard") and I'll rebuild RunDetailBody to match for real, with all the live data wired through.
