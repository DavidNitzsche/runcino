/today hero v3 — strip down

The previous decks tried to make a hero work. David: "none of these tbh. maybe we need a bigger strip down and rethink." So this deck questions whether we need a workout-name hero at all.

First principle Today's workout name is already in the week strip's today tile (the big colored band + DONE check). And the coach voice card below carries the narrative. So duplicating "CRUISE INTERVALS · DONE" big at the top is repetition for repetition's sake. These four directions all kill the hero headline. Differences are how much of a "top strip" remains.
DIRECTION A THIN STATUS BAR · ONE LINE
Thin status bar — ring + meta on one line, week strip carries the day
A single ~70px-tall status row. Ring (medium) + two-line meta (breadcrumb + workout name with green DONE chip). The big "today" energy goes to the week strip below; its today tile already does the loud color story.
