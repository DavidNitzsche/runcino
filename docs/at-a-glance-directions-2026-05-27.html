B · verdict + detail.

each row leads with a one-word verdict, detail follows
Inspired by Whoop's "RECOVERY" verdicts. Each row leads with a colored verdict word (UNDERSLEPT, BUILDING, ON TRACK) on the right, the data follows below. You read the verdict first; if you care, you read the detail. Fastest scan of the four.

C · data viz.

numbers + mini bars/sparklines per row
Each row has the data PLUS a tiny visual — band-position bar for load and sleep, mini sparkline for HRV trend, race-trajectory bar. You can SEE where each metric sits in its band, not just read it. Most information-dense, takes most space.

D · mini tiles.

2×3 grid of self-contained tiles, status dot per tile
Closer to a dashboard pattern. Each metric gets its own tile with big number, label, one-line context, and a status dot in the corner (green/amber/red). Lets the eye pick what matters — the red dots attract attention first, then green dots reassure. Most modern UI pattern.
