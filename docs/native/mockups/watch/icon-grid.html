Design notes:
  • App name on watch home: "faff" (lowercase, 4 chars) · fits the icon label area without truncation
  • Icon background uses the brand orange gradient · matches the race-hero gradient on iPhone for product cohesion
  • Wordmark inside icon (instead of pictogram) keeps it readable + identifiable next to other dense app icons
  • Apple Watch home is honeycomb grid by default · designs work in both grid and list view modes
  • This icon is rendered at 1024×1024 source for App Store · scales to 80×80 for the actual watch render
