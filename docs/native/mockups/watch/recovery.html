WHY COUNT UP

Recovery's relevant question is "how long has this lasted?" + "am I ready?" — the HR drop is the answer. Counting DOWN adds psychological pressure of recovery ending. Counting UP turns recovery into a measurement the runner controls. Planned duration stays visible as the upper bound. "READY" chip appears when HR ≤ readiness threshold (≤130 bpm or 72% of max).

STATE MACHINE
  • State A → State B trigger: HR ≤ readiness threshold for 3 sustained seconds
  • State B doesn't auto-advance. Single soft haptic pulse · runner taps to start next rep early
  • Plan is the upper bound. If runner ignores the chip, watch auto-advances at planned duration (60s)
  • If HR doesn't clear by planned end: rep starts on schedule · summary flags it as "rep N started with elevated HR"
