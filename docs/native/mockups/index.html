How to review

Each mockup shows both light + dark mode side by side. iPhone screens use an iPhone 16 Pro frame (402×874pt). Watch screens use a 41mm Apple Watch frame (324×394pt usable) — designed for the smallest watch size so larger sizes scale up cleanly. Real data throughout (VDOT 46.6, AFC HM Aug 16, max HR 181, RHR 40).

Approve, push back, or call out the specific mockup that needs change. Once locked, mockups become the spec for SwiftUI implementation.

See design-notes.md for typography, spacing, color, and HIG-deviation rationale.
