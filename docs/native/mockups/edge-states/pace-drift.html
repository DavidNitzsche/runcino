DESIGN NOTES · PACE DRIFT v3
  • Tinted background — screen tints amber (#2A1C08) or red (#2A0808). Runner sees "warning state" before reading any number.
  • Same multi-metric stack as work-interval. Pace + HR + Rep timer + footer ribbon. Muscle memory carries through into the drift state — only the colors change.
  • Color migrates to the prescription metric. Pace turns amber → red. HR stays white. Rep timer stays orange (brand anchor).
  • Delta chip in Bebas italic, solid amber/red bg with black text. Pops against the tinted screen.
  • "PUSH IT" on red severity — Bebas italic ALL CAPS at 22pt with 0.10em tracking. The one warm imperative permitted mid-workout.
  • Haptic cadence: subtle amber pulse at sustained 5s · stronger red pulse at sustained 10s · visual reinforces, doesn't replace.
