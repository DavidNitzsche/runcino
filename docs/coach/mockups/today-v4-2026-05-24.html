The principle: cards coach too

  • SLEEP card ends with action, not just a stat. *"Aim for 7.5h tonight to start chipping at the deficit. No need to chase it all back in one night — pick two nights this week to bank an extra hour."* That's coaching, not reporting.
  • UP NEXT card prep advice. *"Keep it conversational. Shake out the long run, nothing more."* — the coach is telling you HOW to run tomorrow, not just WHAT.
  • RACE HORIZON card framing. *"12 weeks gives us room to add real quality. Threshold work starts this week — time to start applying pressure."* — context for why this distance matters now.
  • profile_gap height card. Stands in for the cadence experiment until height is filled. Once height is in, next briefing can prescribe a real target.

What got cut

  • HRmax + RHR profile_gap cards — derived from health_samples. 181 bpm peak across 272 samples; 47 bpm RHR mean across 60 days. No reason to ask the runner.
  • Cadence experiment card — suppressed because height is missing. Coach voice properly defers ("once we have that, we can run a small experiment"). The height gap card carries the call to action.
