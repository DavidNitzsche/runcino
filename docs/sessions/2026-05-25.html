How to test in the morning

5-minute happy path

  1. Open www.faff.run/today on your phone (mobile Safari). Should render the v4 design with coach voice + cards generated from your real prod data.
  2. Tap +Add on the "Your height" card. Enter your height in inches. Watch for the green toast + page reload.
  3. Reload /today?fresh=1 — the briefing should now mention cadence specifically (height-unlocked) and possibly emit a cadence_experiment card with a real target SPM.
  4. Tap a reply chip (SOLID / TIRED / WRECKED). Should toast + fade siblings.
  5. Tap Got it on the HRV fun_fact card. Should toast + fade the card. Future briefings won't show HRV explainer until you clear it.
  6. Open www.faff.run/races. The dark coach-voice block at the top is the v4 layer; the existing race calendar / hero / PRs are below, untouched.

If something looks wrong

  • /today is broken → /overview still works; deep-link there. To debug: hit /api/today?fresh=1 directly and check the JSON payload.
  • The coach voice is off → check web/coach/prompts/daily-briefing.md (the prompt) + web/scripts/test-daily-briefing.mjs (re-run it against current state).
  • A card looks wrong → check the card-specific component in web/app/today/page.tsx + the schema in docs/coach/CARD_LIBRARY.md.
  • Cache is stale → add ?fresh=1 to the URL or the /api/today request to bypass.
  • Want to nuke a card temporarily → edit filterTopicsForRender() in web/coach/today-briefing.ts to filter that kind. Or edit the prompt to not emit it.

Rollback notes (worst case)

  • /today is a new route — deleting web/app/today/ + web/lib/coach/today-state.ts + web/coach/today-briefing.ts + web/app/api/today/ reverts cleanly. /overview untouched.
  • /races coach layer is additive — to revert, remove the import + RacesCoachLayer JSX block in web/app/races/page.tsx. Existing /races content untouched.
  • Schema migrations are idempotent (CREATE IF NOT EXISTS / ADD COLUMN IF NOT EXISTS). To "undo" the new columns, you'd ALTER TABLE DROP COLUMN — but they're orphan-safe; nothing existing reads them.
  • Last green commit before any v4 wiring: 08f0d17 (nav icons fix on v4 mockup).
